以下是对该论文的深度总结：

### 1. 核心问题与研究动机
*   **核心问题**：现有的胸部 X 射线视觉-语言预训练方法在处理“图像-报告”对时存在三个主要缺陷：
    1.  **忽视多视图特性**：大多数方法只关注正视图，或对正侧位视图进行简单的统一融合，忽略了不同视图对病理诊断的差异化贡献。
    2.  **掩码机制效率低**：传统的跨模态掩码语言建模（MLM）采用随机掩码，未能有效利用病理相关的关键视觉信息来引导文本学习。
    3.  **全局对齐的假阴性问题**：在对比学习中，将具有相似病理特征的不同患者视为负样本，会导致特征空间的错误推离（False-negative problem）。

### 2. 方法论
该框架由三个核心模块组成：
*   **关键语义增强的多视图 MLM (KS-MMLM)**：
    *   该模块不再随机掩码，而是通过识别并聚合跨视图（正位和侧位）中与病理相关的关键补丁（Patches）。
    *   利用这些具有丰富语义的视觉特征来指导文本分支的掩码预测，从而强化模型对医学核心术语的理解。
*   **正侧位对齐模块 (FLA)**：
    *   独立提取正位和侧位视图的病理特征，以保留视图特有的细节。
    *   通过一致性约束确保两个视图在语义层面指向相同的诊断结论，减少信息聚合过程中的损失。
*   **高阶语义对齐 (HSA)**：
    *   引入**原型对齐（Prototype-level alignment）**机制。
    *   将图像和文本特征与一组学习到的“语义原型”（代表不同的病理类别或聚类）进行对齐，而非仅仅进行点对点的实例对齐。这有效缓解了具有相同疾病的不同样本被错误区分的问题。
