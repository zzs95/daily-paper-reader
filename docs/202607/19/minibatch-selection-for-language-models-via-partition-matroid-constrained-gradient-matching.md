*   **收敛速度与效率**：随机采样可能选中大量冗余或低信息量的样本，导致训练缓慢。

### 2. 方法论：分区拟阵约束下的梯度匹配
论文提出了一种将**组合优化（拟阵理论）**与**梯度匹配**相结合的方法：
*   **核心思想**：在每一个训练步骤中，不再是随机抽取样本，而是寻找一个样本子集，使其产生的梯度方向和模长最接近“全局梯度”（或代理梯度），同时这个子集的构成必须满足预设的领域分布要求。
*   **关键技术细节**：
    *   **梯度匹配（Gradient Matching）**：通过最小化子集梯度与全集梯度之间的欧氏距离，确保选出的样本对模型参数更新最具代表性。
    *   **分区拟阵约束（Partition Matroid Constraint）**：将数据集按领域划分为不相交的集合。通过拟阵约束，强制要求从每个领域 $i$ 中选择的样本数量 $n_i$ 必须满足 $L_i \le n_i \le U_i$。这保证了模型不会因为追求收敛速度而“偏科”。
