*   **核心问题**：在数字化病理分析中，全切片图像（WSI）具有极高的分辨率（亿级像素），直接处理计算量巨大。目前主流方法是采用多实例学习（MIL），即先用预训练模型提取图像块（Patch）特征，再进行聚合。

    *   **单一模型的局限性**：传统的 ImageNet 预训练 CNN 模型缺乏病理领域的特定知识；而新兴的病理基础模型（PFM）虽然在大规模病理数据上进行了预训练，但在捕获某些局部纹理或特定形态特征时，可能不如针对性设计的 CNN 灵活。
    *   **特征互补性**：研究者认为，基础模型提供的全局语义表征与 CNN 提取的局部特征具有互补性，将两者结合可以更全面地描述复杂的病理组织。

    *   **协同辅助模块**：设计了一种特征融合或注意力引导机制，使两个分支的特征能够相互增强。
    *   **MIL 聚合层**：将融合后的特征输入到注意力池化（Attention-based Pooling）或其他 MIL 聚合器中，生成最终的 WSI 级预测。
