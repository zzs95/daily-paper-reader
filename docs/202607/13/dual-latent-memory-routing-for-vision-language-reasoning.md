### 2. 方法论（Dual-Latent Memory Routing, DLMR）
*   **核心思想**：引入一种“双潜空间”机制，通过构建两条并行的记忆路径来分别处理即时视觉感知和长程视觉记忆，并利用路由机制动态调节两者的权重。
