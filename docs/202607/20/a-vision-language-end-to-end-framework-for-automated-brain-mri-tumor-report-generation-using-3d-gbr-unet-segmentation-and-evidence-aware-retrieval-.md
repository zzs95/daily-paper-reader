    *   独立的语言大模型（LLM）在生成报告时缺乏视觉依据，容易产生“幻觉”（Hallucinations），即生成与影像事实不符的诊断描述。
*   **整体含义**：论文提出了一种整合了 3D 分割、分类与检索增强生成（RAG）的端到端框架，旨在提供可解释、有证据支撑且临床一致的自动化脑肿瘤分析方案。

### 2. 论文提出的方法论
该框架由四个核心模块组成：
*   **3D CNN-LSTM 分类器**：用于提取全局肿瘤特征并进行初步分类（如胶质瘤与其他肿瘤），为后续推理提供高层语义上下文。
*   **GBR UNet（全局瓶颈精炼 UNet）**：
    *   **核心思想**：在 3D UNet 的瓶颈层（Bottleneck）引入轻量级 Transformer 分支。
    *   **技术细节**：采用双路径设计，一条路径使用 3D 卷积保持局部空间精度，另一条路径使用 Transformer 捕捉长程依赖关系。这种设计在保持 CNN 归纳偏置的同时增强了全局上下文建模。
*   **检索增强视觉提示（RAVP）**：
    *   **相似案例检索**：利用视觉嵌入从参考数据库中检索 Top-K 个相似案例。
    *   **多级提示构建**：整合扫描级（全局形态）、实体级（基于分割掩码的定量体积、位置数据）和邻居案例级（相似病例经验）的视觉提示。
*   **架构流程**：输入多模态 MRI $\rightarrow$ 3D 分类与 GBR UNet 分割 $\rightarrow$ 提取定量特征与相似案例 $\rightarrow$ 构造结构化提示词 $\rightarrow$ 输入 Groq 加速的 Llama-3.3-70B 模型生成符合放射学规范的报告。
