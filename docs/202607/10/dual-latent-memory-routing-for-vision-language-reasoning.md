### 2. 论文提出的方法论：Dual-Latent Memory Routing (DLMR)
核心思想是引入一种**双潜空间记忆路由机制**，将视觉特征的处理分为两条互补的路径：

*   **动态路由机制**：在文本生成的每一步，模型通过路由算法动态地从这两个潜空间中检索信息。这种机制允许模型根据当前生成的语境，灵活地在“细节视觉特征”与“高层语义记忆”之间切换。
