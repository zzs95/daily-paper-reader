*   **研究动机**：放射科医生在起草报告时，往往需要根据临床习惯或特定机构要求对报告片段进行修改、补充或规范化。
*   **核心挑战**：目前主流的医学基础模型（如放射报告生成模型）几乎全部采用**自回归（AR）**架构。AR模型只能从左向右逐个生成令牌（token），无法有效利用“右侧上下文”来填充中间缺失的文本（即“任意顺序填充”能力较弱）。
*   **核心问题**：离散扩散语言模型（Discrete Diffusion Language Models）是否能在保持生成质量的同时，凭借其双向去噪的特性，为放射报告提供更高效的交互式起草（如局部修改、中间填空）能力？

### 2. 论文提出的方法论
*   **核心思想**：利用离散扩散模型的**双向去噪机制**。模型不是从左到右预测，而是在一个固定的令牌画布（Canvas）上，通过迭代去噪来生成或修复文本。
*   **关键技术细节**：
    *   **模型架构**：采用 **DiffusionGemma-26B**（混合专家模型 MoE，激活参数约 3.8B），并与其同规模、同血缘的自回归模型 **Gemma-4-26B** 进行严格的对标测试。
    *   **微调策略**：使用 **LoRA（低秩自适应）** 技术。为了保证对比的公平性，两者使用完全相同的视觉编码器（SigLIP）、LoRA 目标（Attention 和共享 MLP 层）以及训练数据。
    *   **任意顺序填充（Any-Order Infill）**：在推理阶段，通过修改采样器，固定画布上的已知片段（Fixed tokens），让模型仅对缺失部分进行去噪。由于扩散模型在每一步都能看到整个画布，因此能同时利用左侧和右侧的上下文。

*   **性能持平或领先**：在医学 VQA 任务中，微调后的扩散模型在所有数据集上均达到或超过了其自回归兄弟模型，且能与 GPT-4.1-mini 等前沿模型竞争。

*   **临床实用性强**：提出的“任意顺序填充”非常契合放射科医生修改报告的真实工作流。
*   **评估严谨**：采用了鲁棒的 LLM 裁判机制，避免了传统 NLP 指标在医学文本上的失真。

### 8. 不足与局限
*   **任务覆盖面**：实验主要集中在医学 VQA 和胸部 X 光（CXR）报告上，尚未验证在更复杂的 3D 影像（如 CT/MRI 序列报告）中的表现。
