*   **核心问题**：当前多模态大语言模型（MLLM）在**训练效率**与**推理效率**之间存在难以调和的矛盾。
*   **背景分析**：
    *   **嵌入空间对齐架构（如 LLaVA）**：将视觉特征转换为视觉 Token 并拼接到文本序列中。这种方式训练效率高（利用了 LLM 的自回归特性），但推理效率低，因为视觉 Token 占据了极长的上下文窗口，导致计算量随序列长度呈平方级增长。
    *   **交叉注意力架构（如 Flamingo）**：通过交叉注意力机制融合视觉信息。这种方式推理效率较高（视觉 Token 不占用文本 KV Cache），但训练极其缓慢且显存占用高，因为需要处理复杂的注意力交互。
*   **研究目标**：提出一种新架构，旨在同时实现高效的训练过程和快速的推理响应。
