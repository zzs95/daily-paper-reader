### 1. 核心问题与整体含义（研究动机和背景）
在 LLM 预训练中，不同领域（如代码、科学论文、网页等）的数据混合比例决定了模型的最终能力。
*   **现状与痛点**：目前的静态混合策略（如固定比例）无法适应模型在不同训练阶段的演进需求；而全动态在线调整策略虽然灵活，但计算开销极其昂贵，难以扩展到大规模预训练。
*   **核心目标**：寻找一种“半动态”方案，既能根据模型当前状态快速计算最优数据配比，又能保持极低的计算开销，且在较长的训练区间内持续有效。

### 2. 方法论：核心思想与关键技术
TiKMiX 的核心思想是利用**数据影响力（Data Influence）**来量化不同领域对模型性能的贡献。
*   **Group Influence（群体影响力）**：论文提出了一种高效的度量方法。它不计算单个样本的影响力，而是通过梯度累积技术计算整个数据域的影响力。利用 EKFAC（经验克罗内克因子分解）近似 Hessian 矩阵的逆，将计算复杂度从样本量级降低到领域量级。
*   **TiKMiX-D（直接优化）**：将数据混合建模为一个约束非线性优化问题。目标函数平衡了三个维度：最大化总影响力、保持各任务间影响力的均匀性、以及通过熵约束维持数据多样性。
*   **TiKMiX-M（混合模型优化）**：考虑到数据域之间可能存在非线性交互，TiKMiX-M 引入了回归代理模型（如 LightGBM）。它通过少量采样训练一个预测器，预测不同配比下的总影响力，并利用迭代搜索算法寻找全局最优配比。
