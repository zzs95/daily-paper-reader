### 1. 论文的核心问题与整体含义
论文聚焦于**多智能体搜索系统（MASS）**的端到端优化问题。
*   **研究动机**：虽然大语言模型（LLM）通过多智能体协作和检索增强生成（RAG）可以解决知识过时和复杂推理问题，但现有的优化手段存在缺陷。手动 Prompt 工程和监督微调（SFT）成本高且难以适应动态需求；而传统的多智能体强化学习（如 MAPPO）依赖庞大的 Critic 网络，导致训练不稳定且显存开销巨大。
*   **核心问题**：如何在不依赖 Critic 网络的情况下，高效地优化具有不同角色的多个智能体，使其能够协同实现全局系统目标，并解决智能体之间复杂的间接依赖和信用分配问题。

### 2. 论文提出的方法论：MHGPO
论文提出了**多智能体异构群体策略优化（MHGPO）**框架，这是首个针对 MASS 的基于群体的优化算法（GOA）。
*   **核心思想**：取消 Critic 网络，通过在“异构群体”中估计相对奖励来计算优势（Advantage），将多智能体学习转化为多任务优化问题。
*   **关键技术细节**：
    *   **参数共享**：所有智能体角色（如重写者、重排者、回答者）共享同一个 LLM 主干模型，降低内存开销。
    *   **反向奖励传播**：将系统最终的奖励（如回答的 F1 分数）沿执行轨迹反向传播并聚合，使上游智能体能感知其对下游行为的影响，捕捉间接依赖。
    *   **异构群体优势估计**：不同于单智能体 GRPO 的同质化比较，MHGPO 在同一根查询下，对比不同中间决策产生的异构轨迹，从而将优化重心从局部表现转向全局成功。
*   **采样策略**：
    *   **IS（独立采样）**：每个智能体独立生成同质组，作为基准。
    *   **FoF（首位分叉）**：仅在第一个智能体处分叉生成多个轨迹，后续智能体执行 1 对 1 采样，效率最高。
    *   **RR（轮询分叉）**：随机选择分叉点，平衡全局协调压力与局部学习稳定性。
