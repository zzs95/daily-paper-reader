### 1. 论文的核心问题与整体含义
*   **研究动机**：当前的放射科报告生成（RRG）评估协议（如 BLEU、CheXpert F1、GREEN 等）主要关注生成文本与参考报告之间的词汇重叠或临床事实一致性。
*   **核心问题**：这些指标无法检测模型是否真正“看到”了病灶。模型可能通过学习数据集中的**视觉捷径（Vision Shortcuts）**（如利用流行病学先验、采集伪影或病灶间的共现关系）来生成看似正确的报告，而非基于真实的影像证据。
*   **研究目标**：提出一个名为 **SHOVIR** 的基准测试，旨在量化视觉语言模型（VLM）在生成诊断陈述时对视觉证据的真实依赖程度。

### 2. 论文提出的方法论
SHOVIR 通过受控的**遮蔽实验（Occlusion Experiments）**来探测模型的决策逻辑：
*   **数据增强与对齐**：将 MIMIC-CXR 和 PadChest-GR 数据集中的空间边界框（Bounding Boxes）与 CheXpert 疾病标签进行映射，确保每个病灶区域都有明确的类别标注。
*   **图像级遮蔽（Image-Level）**：
    *   **Full Noise Image**：用保持原始图像统计特性（均值/方差）的高斯噪声替换整幅图像，测量模型性能的下降程度（$\Delta F_N$）。
*   **疾病级遮蔽（Disease-Level）**：
    *   **Object Class Occlusion (OCO)**：仅遮蔽目标病灶区域。若模型仍能诊断该病，说明存在**直接捷径**。
    *   **Different Object Class Occlusion (DOCO)**：遮蔽共现的其他病灶区域，保留目标病灶。若模型诊断能力下降，说明存在**上下文捷径**（依赖共现关系而非目标本身）。
    *   **Random Occlusion (RO)**：作为对照组，遮蔽随机的非病灶区域。
*   **核心指标**：定义了 $\Delta OCO$ 和 $\Delta DOCO$。高 $\Delta OCO$ 表示良好的视觉定位；高 $\Delta DOCO$ 则暴露了模型对上下文背景的过度依赖。

*   **性能与定位的脱节**：在基准测试中获得最高临床准确分数的模型（如 MedGemma, MAIRA-2），在空间定位（$\Delta OCO$）上的表现并不一定最好。这意味着**临床流利的报告可能掩盖了模型对视觉证据的浅层依赖**。
