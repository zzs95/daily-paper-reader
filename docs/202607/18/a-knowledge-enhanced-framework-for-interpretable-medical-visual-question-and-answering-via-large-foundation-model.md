### 1. 核心问题与整体含义（研究动机和背景）
医学视觉问答（MVQA）旨在让计算机能够根据医学图像回答临床问题。尽管目前的大语言模型（LLM）和多模态基础模型在通用领域表现优异，但在医学领域面临两大挑战：
*   **领域知识匮乏**：通用模型往往缺乏深度的医学专业知识，容易在处理复杂病理图像时产生“幻觉”。
*   **缺乏可解释性**：医疗决策关乎生命，黑盒模型给出的答案难以让医生信服。
**研究动机**：通过引入外部医学知识库来增强基础模型，使其不仅能给出准确答案，还能提供逻辑清晰的诊断依据（Rationale），从而提升医疗AI的透明度和可信度。

### 2. 论文提出的方法论
该论文提出了一种**知识增强的可解释框架**，其核心流程如下：
*   **知识检索模块**：从输入的医学图像和问题中提取关键实体，利用医学知识图谱（如 UMLS、SemMedDB）或医学文献数据库检索相关的病理学、解剖学知识。
*   **多模态特征融合**：采用视觉编码器（如 CLIP 或 PMC-CLIP）提取图像特征，并与问题文本、检索到的外部知识进行对齐。
*   **基于基础模型的推理引擎**：利用预训练的医学大模型（如 PMC-LLaMA 或 BLIP-2 架构）作为核心推理器。
*   **可解释性生成**：模型被设计为输出“链式思考”（Chain-of-Thought），即先生成基于知识的推理过程（Rationale），再给出最终答案。
