本文针对医学图像分割中一个极具挑战性的现实场景：**开集无源主动领域自适应（Active Source-free Open-set Domain Adaptation, ASFOSDA）**。
*   **研究动机**：传统的领域自适应（DA）假设源域和目标域类别相同（闭集），但在临床实践中，目标域常包含源域未见的新解剖结构或病理（开集）。
*   **核心挑战**：1) **数据隐私**导致无法访问源域数据（无源）；2) **负迁移**风险，即模型可能将目标域的未知类别错误对齐到源域已知类别；3) **标注成本**高昂，需要通过主动学习（AL）以极小的标注代价实现最大性能提升。

### 2. 论文提出的方法论
论文提出了一种结合**主动开集查询策略**与**目标精炼自训练**的框架：

*   **核心思想**：通过量化样本的“不确定性”和“代表性”来筛选最具信息量的目标域样本，并利用半监督学习进一步挖掘未标注数据。
*   **关键技术细节**：
    1.  **类感知分解不确定性 (CDU)**：利用测试时增强（TTA）将不确定性分解为：
        *   **类感知偶然不确定性 (CAU)**：计算原始与增强图像在未知类别上的预测熵均值，衡量模型对目标模式建模的可靠性。
        *   **类感知认知不确定性 (CEU)**：计算预测能量（Energy）的方差，衡量模型跨域泛化的能力。
    2.  **类无关原型差异 (CPD)**：
        *   **跨域差异 (CDD)**：计算高不确定性样本与低不确定性样本（代表源域知识的原型）之间的语义距离。
        *   **自域差异 (SDD)**：计算高不确定性样本之间的语义差异，以减少冗余，确保选出的样本具有多样性。
    3.  **目标精炼自训练 (Target-refined Self-training)**：先用标注样本训练模型，再用该模型为未标注样本生成高质量伪标签，最后进行半监督联合训练。
