*   **研究动机**：尽管 2D 视觉基础模型（如 DINO, CLIP）在自然图像领域取得了巨大成功，但在 3D 医学影像（尤其是 CT）中的应用受限。
*   **核心挑战**：
    *   **强度变异性**：CT 的亨氏单位（HU）代表物理密度而非颜色，不同窗口设置（如肺窗 vs 软组织窗）会改变解剖结构的可见性。
    *   **各向异性与层厚**：切片厚度不一会导致部分体积效应，引入系统性偏差。
    *   **空间复杂性**：3D 容积数据具有极高的内存占用和严重的类别不平衡（如巨大解剖结构中的微小结节）。
*   **整体含义**：本研究旨在开发一种原生支持 3D CT 数据的自监督基础模型 **CoralBay**，通过学习丰富的空间表示来捕捉全局语义和精细局部结构，从而在多种放射学下游任务中实现高效迁移。

### 2. 方法论：核心思想与关键技术
*   **核心框架**：基于 **DINO** 的自监督自蒸馏（Self-distillation）范式，并将其从 2D 扩展到原生 3D 领域。
*   **骨干网络**：采用分层式 **3D Swin Transformer**。利用其移动窗口自注意力机制（Shifted-window self-attention）来高效建模长程空间依赖，同时控制计算成本。
*   **多尺度特征融合**：
    *   通过 3D 自适应平均池化将不同阶段的特征图转化为固定长度表示并进行拼接。
    *   这种设计使模型能同时编码全局解剖语义（如器官形状）和精细局部结构（如血管边界），避免了 DINOv2 中复杂的 iBOT 掩码损失。
*   **放射学专用增强**：
    *   **3D 裁剪策略**：从 CT 卷中提取 2D 无法实现的全局（$96^3$）和局部（$48^3$）3D 裁剪块。
    *   **HU 窗口增强**：在预训练中随机采样临床相关的 HU 窗口（如肺、肝、脑、腹部等），强制模型学习对可视化协议具有不变性的特征。
*   **推理技术**：采用滑动窗口技术处理全扫描影像，通过拼接和池化生成扫描级表示。
