*   **核心挑战**：目前大多数研究集中在如何“去噪”，但获取临床配对的（同病人、同部位、标准剂量与低剂量）数据极其困难且不符合伦理。此外，直接在标准剂量CT（SDCT）上训练的模型在面对真实LDCT数据时往往表现不佳（领域偏移）。
*   **研究目标**：本研究反其道而行之，探索如何将现有的SDCT图像“退化”为高质量的合成LDCT图像，以此作为数据增强手段，训练出对噪声更具鲁棒性的肺结节放射组学分类模型。

### 2. 方法论
核心思想是对比三种不同的图像退化技术，生成合成LDCT数据用于训练分类器。
*   **方法1：简单正弦图域模拟（Sinogram-domain Simulation）**

*   **方法2：基于物理模型的Pix2Pix复制（Physics-based via Pix2Pix）**

*   **方法3：非配对图像生成的CycleGAN**
