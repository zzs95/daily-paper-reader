### 1. 核心问题与研究背景
*   **研究动机**：胸部 X 射线（CXR）诊断通常依赖于正位（Frontal）和侧位（Lateral）多个视角的影像及其对应的放射学报告。
*   **现有挑战**：
    1.  **多视图利用不足**：大多数现有 VLP 方法仅关注单视图，或在融合多视图时采用统一权重，忽略了不同视角对病理诊断的贡献差异。
    2.  **掩码机制盲目性**：传统的掩码语言建模（MLM）采用随机掩码，缺乏跨模态语义引导，导致模型难以学习到深层的图文关联。
    3.  **全局对齐的假阴性问题**：在对比学习中，语义相似的不同样本对常被错误地视为负样本，限制了表征的泛化能力。

### 2. 方法论
论文提出了一个包含三个核心组件的预训练框架：
*   **语义感知多视图 MLM (Key Semantics-enhanced Multi-view MLM)**：
    *   该模块不再随机掩码文本，而是通过识别并聚合跨视图中与病理相关的关键图像补丁（Patches），利用这些富含语义的视觉特征来指导文本掩码的恢复，增强跨模态交互。
*   **正侧位对齐模块 (Frontal-Lateral Alignment, FLA)**：
    *   独立提取正位和侧位影像的病理特征。在聚合过程中，该模块确保两个视图之间的语义一致性，同时保留各自特有的关键信息，避免了简单融合导致的信息丢失。
*   **高阶语义对齐 (High-order Semantic Alignment, HSA)**：
    *   引入了基于“原型”（Prototypes）的聚类对齐机制。通过将图像/文本特征与语义一致的聚类中心（原型）进行对齐，从高阶语义层面缓解了实例级对比学习中的假阴性问题。
