### 1. 核心问题与整体含义（研究动机和背景）
在组织病理学图像分析中，获取像素级的分割标注（Pixel-level annotations）极其耗时且需要极高的专业知识，导致标注成本高昂。因此，仅利用**图像级标签**（Image-level labels）实现弱监督分割（WSHS）具有重要的临床意义。
然而，病理图像面临一个核心挑战：**类间同质性（Inter-class homogeneity）**。即不同类别的组织在纹理、颜色和形态上可能高度相似，导致传统的弱监督方法（如基于类激活图 CAM 的方法）难以准确区分边界，容易产生误判。

### 2. 方法论：核心思想与关键技术
论文提出了**层级化提示对比学习（Hierarchical Prompt Contrastive Learning, HPCL）**框架，旨在通过多层级的特征约束来增强类别的判别性。

*   **核心思想**：利用“提示（Prompt）”机制引导模型关注特定类别的特征，并结合“对比学习”在不同特征层级上拉开不同类别间的距离，缩小同类别的分布差异。
*   **关键技术细节**：
    *   **层级化提示机制（Hierarchical Prompting）**：在骨干网络的不同深度引入可学习的提示向量。这些提示向量充当了类别特征的“锚点”，帮助模型在从浅层（纹理）到深层（语义）的特征提取过程中，始终保持对目标类别的敏感度。
    *   **提示对比学习（Prompt Contrastive Learning）**：通过构建正负样本对，在特征空间中进行对比。具体而言，将同一类别的提示特征聚合，并使其在投影空间中相互靠近，而将不同类别的特征推离。
    *   **多尺度特征融合**：为了应对病理组织大小不一的问题，HPCL 结合了多尺度的特征表达，确保微小的病变区域和宏观的组织结构都能被准确捕捉。
    *   **流程**：输入图像 -> 骨干网络（带层级提示） -> 生成初步热力图 -> 对比损失约束 -> 细化生成像素级掩码。
