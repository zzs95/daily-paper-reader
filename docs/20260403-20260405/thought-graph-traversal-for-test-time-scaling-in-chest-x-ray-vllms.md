### 2. 论文提出的方法论：思维图遍历（Thought Graph Traversal）
*   **核心思想**：将单一的线性推理过程扩展为结构化的“思维图”搜索。模型不再只生成一个答案，而是生成多个可能的推理分支，并通过图遍历算法寻找最优路径。
*   **关键技术细节**：
    1.  **思维生成（Thought Generation）**：针对输入的 CXR 图像，模型生成多个中间推理步骤（如观察征象、解剖定位、鉴别诊断）。
    2.  **图构建**：将这些推理步骤作为节点，逻辑流向作为边，构建出一个思维图。
    3.  **验证与评分（Verification）**：引入一个验证机制（可以是模型自评或小型专用评估器），对每个推理节点的逻辑合理性和医学准确性进行打分。
    4.  **遍历策略**：采用如束搜索（Beam Search）或最佳优先搜索（Best-first Search）等算法，在图中搜索累计得分最高的路径，作为最终的诊断报告。
*   **算法流程**：输入图像 -> 多路径思维采样 -> 节点评分 -> 路径搜索与剪枝 -> 输出最终推理链及结论。
