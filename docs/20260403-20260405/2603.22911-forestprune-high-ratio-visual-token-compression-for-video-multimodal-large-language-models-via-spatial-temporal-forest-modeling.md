### 1. 核心问题与研究动机
*   **核心问题**：视频多模态大语言模型（Video MLLMs）在处理长视频或高分辨率视频时，视觉 Token 数量呈爆炸式增长，导致巨大的计算开销和内存占用。
*   **研究动机**：现有的 Token 压缩方法（如剪枝或合并）多源于图像任务，在处理视频时往往只关注单帧内的重要性，忽略了视频在时间维度上的连续性和冗余性。这导致在**高压缩比例（如 90% 以上）**下，现有方法会因丢失关键时空信息或保留过多跨帧冗余 Token 而导致性能大幅下降。

### 2. 方法论：ForestPrune 时空森林建模
ForestPrune 是一种**无需训练（Training-free）**的即插即用方法，其核心思想是将视频 Token 建模为跨越时空的“森林”结构：
*   **候选节点提取**：首先利用现有的轻量级方法（如随机采样或简单的图像剪枝）获取每帧的代表性 Token 作为森林的候选节点。
*   **时空森林构建**：
    *   **语义与空间约束**：计算 Token 间的余弦相似度（语义）和坐标距离（空间）。只有当两个 Token 语义高度相似且空间位置接近时，才考虑建立连接。
    *   **时间顺序约束**：连接必须遵循时间先后顺序（从前一帧指向后一帧）。
    *   **树结构生成**：通过定义的连接矩阵识别“根节点”（Root Nodes），并利用 `LinkNode` 算法递归地将相似的 Token 串联成“树”。
*   **全局剪枝策略**：
    *   **重要性评估**：根据树的深度（Depth）和节点角色（Role）评估重要性。通常，较深的树代表该视觉特征在视频中持续时间长、稳定性高。
    *   **优先级剪枝**：优先剪除“叶子节点”（末端）和“短树”，保留“根节点”和“主干节点”。这种策略确保了在极高压缩率下，视频的全局语义骨架得以保留。
