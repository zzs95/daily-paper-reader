### 2. 论文提出的方法论
*   **核心思想**：将关系型数据库中的行记录转化为序列文本，利用**预训练语言模型（PLM）**强大的语义建模能力，将缺失值插补任务转化为一个“完形填空”或“序列生成”任务。
*   **关键技术细节**：
    *   **数据序列化**：将数据库的元数据（列名）与属性值结合，构建成自然语言描述的句子。
    *   **语义感知建模**：利用 PLM（如 BERT、RoBERTa 等架构）学习属性之间的语义依赖。例如，通过上下文推断出“职业”列缺失值应为“工程师”。
    *   **动态适应机制**：针对动态数据库，模型能够通过微调（Fine-tuning）或提示学习（Prompt Learning）快速捕捉新出现的语义模式。
