### 1. 核心问题与研究背景
*   **核心问题**：放射科报告生成（RRG）的主要挑战在于如何实现**复杂的视觉特征**与**具有层级结构的医学报告**之间的精细对齐。
*   **研究背景**：现有的方法通常将报告视为扁平的单词序列，忽略了报告中存在的“段落-句子-单词”这种天然的语义层级。这种简化导致模型难以捕捉到视觉病灶与临床描述之间细粒度的对应关系，从而限制了生成报告的准确性和临床一致性。

### 2. 方法论：RIHA 框架
RIHA 的核心思想是在多个粒度级别上同步对齐图像和文本特征。
*   **视觉特征金字塔 (VFP)**：利用卷积神经网络或 Transformer 骨干网络提取多尺度的视觉特征，捕捉从全局解剖结构到局部细微病灶的视觉信息。
*   **文本特征金字塔 (TFP)**：构建多粒度的文本表示，分别对应报告的段落级（全局语义）、句子级（特定诊断）和单词级（核心术语）。
*   **跨模态层级对齐 (CHA) 模块**：这是核心创新点。该模块利用**最优传输（Optimal Transport, OT）**算法，在段落、句子和单词三个层级上分别计算视觉与文本特征的匹配成本，从而实现更稳健的跨模态映射。
*   **相对位置编码 (RPE)**：在解码器中引入 RPE，以更好地建模 Token 之间的空间和语义关系，增强生成文本时的局部逻辑连贯性。
