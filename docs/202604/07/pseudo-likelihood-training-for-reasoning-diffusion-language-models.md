### 1. 论文的核心问题与整体含义（研究动机和背景）
当前大语言模型（LLM）提升推理能力的核心技术是**策略梯度强化学习（PGRL）**（如 PPO 或 GRPO）。然而，这些方法在**扩散语言模型（dLLM）**上难以直接应用。
*   **技术瓶颈**：PGRL 需要计算生成序列的精确对数似然（Log-Likelihood），而 dLLM 的生成过程是非顺序、非因子化的迭代去噪过程，计算精确似然在计算上极其昂贵且不切实际。
*   **现有缺陷**：以往尝试将 RL 应用于 dLLM 的方法要么不可扩展，要么使用了缺乏理论依据的近似（如偏差较大的均值场近似），导致训练不稳定或无法收敛到最优解。
*   **研究目标**：提出一种理论严谨、计算高效的框架，使扩散模型也能像自回归模型一样通过强化学习实现对齐和推理能力的增强。

### 2. 论文提出的方法论：PADRE
论文提出了 **PADRE**（Pseudo-Likelihood Alignment for Diffusion Reasoning Models）框架，其核心思想是**将 RL 对齐问题转化为伪似然匹配问题**。

*   **核心思想**：基于 Hammersley-Clifford 定理，如果两个正分布的所有单 token 条件概率分布相等，则它们的联合分布也相等。因此，PADRE 不去匹配难以计算的联合概率，而是匹配**条件边缘分布**。
*   **关键技术细节**：
    *   **伪似然目标函数**：定义了一个基于条件 KL 散度的目标函数 $J_{PKL}$，它避免了计算配分函数（Partition Function），且在理论上与 KL 正则化强化学习具有相同的全局最优解。
    *   **高效多条件评估**：为了避免为每个 token 位置都进行一次前向传播，作者设计了一种**增强输入（Augmented Input）**技术。通过在 Prompt 后拼接带有共享位置编码的 Mask 序列，并配合特殊的**注意力掩码（Attention Mask）**，可以在单次前向传播中同时估计序列中所有位置的条件概率。
    *   **变体策略**：提出了 **PKL-U**（块均值场近似）和 **PKL-T**（基于生成轨迹的近似）两种高效计算变体，以平衡计算成本与估计精度。
    *   **算法流程**：采样 Prompt -> 生成多个回复 -> 计算奖励并标准化（GRPO 风格）-> 构造扰动 Prompt -> 通过伪似然梯度更新模型。

### 6. 论文的主要结论与发现
*   **性能领先**：PADRE 在所有数学基准测试中均达到或超过了现有 dLLM 训练基线。例如，在 GSM8K 上达到 87.3% 的准确率，显著优于 d1（82.1%）。
*   **代码能力提升**：在 HumanEval 和 MBPP 上，PADRE 同样展现了比底座模型和 d1 方法更强的生成能力。

*   **算力效率**：虽然 PADRE 的计算量略高于简单近似方法，但通过单次前向传播技巧，其在相同 FLOPs 预算下的最终表现优于竞争对手。

### 7. 优点：亮点与创新

*   **计算巧妙**：通过共享位置编码和注意力掩码实现“一图多算”，极大地提高了条件概率的评估效率。
*   **通用性**：该框架不依赖于特定的扩散调度（Schedule），具有推广到其他离散扩散模型的潜力。

### 8. 不足与局限
*   **长链推理挑战**：在需要极长步骤推理的任务（如复杂数独）中，条件概率的近似误差可能会累积，论文对此类极端任务的覆盖有限。
