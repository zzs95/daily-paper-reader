*   **研究动机**：GUI 定位（Grounding）是 GUI 智能体的核心能力，要求模型根据自然语言指令在屏幕截图中精准找到目标元素并预测动作。
*   **现状与局限**：目前该领域由自回归（AR）模型主导。虽然 AR 模型性能强劲，但受限于单向注意力和顺序解码。
*   **核心问题**：具有双向注意力、并行生成和迭代优化特性的“离散扩散视觉-语言模型（DVLM）”是否能成为 GUI 定位的有效替代方案？

### 2. 方法论
论文基于 **LLaDA-V**（一种离散扩散模型）进行了适配，提出了针对 GUI 任务的改进方案：
*   **核心思想**：将 GUI 定位建模为从多模态输入（图像+指令）生成文本（动作+坐标）的过程，并利用扩散模型的迭代去噪能力提升坐标预测的精度。
*   **混合掩码调度（Hybrid Masking Schedule）**：这是本文的关键创新，旨在捕捉边界框坐标的几何层级结构。
    *   **线性掩码阶段（Linear Masking）**：采用标准扩散模型的随机掩码策略，让模型学习预测动作类型和锚点坐标 $(x_1, y_1)$。
    *   **全确定性掩码阶段（Full Deterministic Masking）**：在训练中强制掩码所有响应 Token，专门强化模型在已知锚点条件下预测边界框范围 $(x_2, y_2)$ 的能力。
*   **推理策略**：采用低置信度重掩码（Low-confidence re-masking），在迭代过程中保留高置信度预测，重新生成不确定的部分。
