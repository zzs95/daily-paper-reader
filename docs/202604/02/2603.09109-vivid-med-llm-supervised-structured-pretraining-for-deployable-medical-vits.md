### 1. 核心问题与研究背景
*   **核心问题**：现有的医疗影像预训练方法主要依赖 **one-hot 标签**（忽略了疾病间的临床相关性，如胸腔积液与肺水肿的共存关系）或 **自由文本**（语义表达过于多变且模糊）。
*   **研究动机**：大型语言模型（LLM）内部嵌入了结构化的临床知识。本文旨在探索如何利用 LLM 作为“老师”，将这种结构化的语义知识蒸馏到轻量级的视觉 Transformer（ViT）中，从而在不增加推理成本的前提下，提升医疗影像模型的泛化能力和部署效率。

### 2. 方法论
VIVID-Med 框架由三个核心组件构成：
*   **统一医学图谱（UMS）监督**：
    *   将临床发现转化为可验证的 **JSON 字段-状态对**（如 `{"肺炎": "存在", "心脏肥大": "不确定"}`）。
    *   **可回答性感知掩码（Answerability-Aware Masking）**：通过掩码过滤掉无法评估的临床发现，使模型专注于有意义的信号，避免噪声梯度。
    *   **字段查询训练**：对长尾（低频）临床发现进行过采样，增强模型对罕见病的识别能力。
*   **结构化预测分解（SPD）投影器**：
    *   采用类似 Q-Former 的结构，将 ViT 提取的特征分解为 $G$ 个互补的语义组。
    *   **正交正则化（Orthogonality Regularization）**：通过损失函数强制不同查询组关注图像中不同的解剖区域（如心影、肺野等），确保提取特征的互补性。
*   **训练与部署流程**：
    *   **训练阶段**：冻结 LLM（如 Qwen2.5），通过“教师迫导（Teacher Forcing）”优化 ViT 编码器和 SPD 投影器。
    *   **部署阶段**：**完全丢弃 LLM 和 SPD 投影器**，仅保留轻量级的 ViT 主干网络，实现高效部署。
