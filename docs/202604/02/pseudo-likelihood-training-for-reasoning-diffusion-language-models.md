### 论文总结：PADRE - 推理扩散语言模型的伪似然训练

#### 1. 论文的核心问题与整体含义（研究动机和背景）
当前大语言模型（LLM）的对齐和推理能力提升主要依赖于**策略梯度强化学习（PGRL）**，如 PPO 或 GRPO。然而，这些方法在**扩散语言模型（dLLM）**上难以应用。
*   **核心矛盾**：PGRL 需要计算生成序列的精确对数似然（Log-Likelihood），这在自回归模型中很容易实现，但在 dLLM 中计算成本极高且难以处理。
*   **现有缺陷**：目前尝试将 RL 应用于 dLLM 的方法要么不可扩展，要么使用了缺乏理论依据的粗糙近似（如单步均值场近似），导致训练不稳定或无法收敛到最优解。
*   **研究目标**：提出一种理论严谨、计算高效的框架，使扩散模型也能利用强化学习进行推理任务的对齐。

#### 2. 论文提出的方法论：PADRE
论文提出了 **PADRE**（Pseudo-Likelihood Alignment for Diffusion Reasoning Models）框架，其核心思想是将 RL 对齐问题转化为**伪似然（Pseudo-Likelihood）匹配问题**。

*   **核心思想**：基于 Hammersley-Clifford 定理，如果两个正分布的所有单 token 条件概率（即 $p(y_i | y_{-i})$）相等，则它们的联合分布也相等。因此，PADRE 不去计算难以获取的联合似然，而是去匹配模型与目标分布的**条件边际分布**。
*   **关键技术细节**：
    *   **目标函数**：定义了一个“伪 KL 散度”目标，通过最小化模型条件分布与由奖励函数加权的参考分布之间的差异。
    *   **梯度等价性**：证明了该伪似然目标的梯度在优化终点上与标准的熵正则化强化学习（RLHF）是一致的。
    *   **高效评估技巧**：为了避免为每个 token 位置都进行一次前向传播，作者利用 Transformer 的位置编码特性，构造了增强输入序列（Prompt + Completion + Masks），通过特定的注意力掩码（Attention Mask）在单次前向传播中同时估计多个位置的条件概率。
    *   **变体方案**：提出了 **PKL-U**（块均值场近似）和 **PKL-T**（基于生成轨迹的近似）以平衡计算效率与估计精度。

#### 3. 实验设计
*   **基准模型**：以最近开源的高性能扩散语言模型 **LLaDA-8B** 为基础。
*   **数据集/场景**：
    *   **数学推理**：GSM8K（小学数学）、MATH（竞赛级数学）、MATH500（精选子集）。
    *   **代码生成**：HumanEval、MBPP。
*   **对比方法（Baselines）**：
    *   自回归模型对齐方法：Dream 7B。
    *   扩散模型对齐方法：diffu-GRPO、d1-LLaDA（基于均值场近似）、VRPO（方差缩减优化）、wd1（加权策略优化）。

#### 4. 资源与算力
*   **硬件环境**：使用了 Lambda Labs 的服务器，配置为 **8 张 NVIDIA A100-80G GPU**。
*   **训练细节**：使用 AdamW 优化器，余弦学习率衰减，初始学习率 5e-6，训练步数为 10k 步。
*   **数据增强**：在训练中对 Prompt 进行了 15% 概率的随机掩码（Token Perturbation）以增强泛化性。

#### 5. 实验数量与充分性
*   **实验规模**：论文在 5 个主流推理基准数据集上进行了测试，涵盖了数学和代码两大核心推理领域。
*   **消融实验**：对比了 PKL（全量）、PKL-U、PKL-T 以及带扰动的 PKL+ 等多个变体，验证了不同近似策略对性能的影响。
*   **客观性评价**：实验设计较为充分，不仅对比了最终准确率，还通过 KL 散度分析了不同近似方法与真实分布的偏差，并提供了 FLOPs（算力消耗）调整后的性能对比曲线，确保了对比的公平性。

#### 6. 论文的主要结论与发现
*   **性能领先**：PADRE 在所有测试的数学和代码基准上均达到或超过了现有的 dLLM 训练基准。例如，在 GSM8K 上达到 87.3% 的准确率，显著优于 d1-LLaDA 的 82.1%。
*   **理论验证**：实验证实了单步均值场近似（如 d1 方法）存在较大的分布偏差，而基于伪似然的条件概率估计能更准确地捕捉模型分布。
*   **稳定性**：PADRE 提供了一个比直接应用 PGRL 更稳定的训练目标，解决了扩散模型在 RL 过程中的梯度偏差问题。

#### 7. 优点
*   **理论严谨性**：通过伪似然理论为扩散模型的 RL 训练提供了坚实的数学基础，避免了盲目近似。
*   **计算创新**：巧妙利用 Transformer 的注意力掩码机制实现了多位置条件概率的高效并行计算，解决了计算瓶颈。
*   **通用性**：该框架不依赖于特定的扩散调度，可广泛应用于各类掩码扩散语言模型。

#### 8. 不足与局限
*   **扩展性挑战**：虽然提出了高效计算技巧，但在处理极长序列时，伪似然的计算开销仍高于自回归模型的逐 token 似然计算。
*   **任务覆盖面**：实验主要集中在数学和代码，对于需要极长链推理或具有复杂约束的任务（如 Sudoku 求解），扩散模型的表现仍有待进一步验证。
*   **近似偏差**：尽管比前人方法更准，但 PKL-U 和 PKL-T 依然是某种形式的近似，在极端情况下可能仍存在理论最优解的偏移。
