*   **报告生成（Report Generation）**：引入了**编码器-解码器（Encoder-Decoder）架构**。
    *   **编码器**：提取图像的视觉特征。
    *   **解码器**：通常采用循环神经网络（RNN/LSTM）或 Transformer 架构，将视觉特征映射为自然语言描述，生成符合放射科规范的诊断报告。
