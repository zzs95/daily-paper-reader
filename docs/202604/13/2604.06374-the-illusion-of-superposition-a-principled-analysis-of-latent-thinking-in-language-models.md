*   **研究动机**：近年来，隐式推理（Latent CoT，即在连续向量空间而非离散文本空间进行推理）被认为优于传统的离散思维链。理论界假设其核心优势在于**“叠加”（Superposition）**——模型能在单个表示中同时维持多个候选解，避免了离散 token 必须过早做出选择的局限。
*   **核心问题**：大语言模型（LLM）在实际进行隐式推理时，是否真的利用了这种“叠加”能力？还是这仅仅是一种理论上的幻觉？

### 2. 论文提出的方法论
论文通过三种不同的实现制度（Regimes）来系统性地分析叠加现象：
*   **免训练制度（Training-free / Soft Thinking）**：直接将输入嵌入构造为词表 token 嵌入的凸组合（加权平均），观察现成模型如何处理这种“人工叠加”的输入。
*   **微调制度（Fine-tuned / Coconut）**：遵循 Coconut 方法，通过课程学习将离散 CoT 逐步替换为连续的隐式状态，使模型学会生成隐式思维。
*   **从头训练制度（From-scratch）**：在特定任务（如 ProsQA）上从零开始训练小型 GPT-2 架构模型，使其完全依赖隐式 token 解决问题。
*   **关键分析技术**：
    *   **Logit Lens**：将中间层的隐藏状态投影回词表空间，观察预测概率分布的演变。
    *   **实体级探测（Entity-level Probing）**：针对合成任务，分析模型在推理步骤中对“正确下一步”、“错误邻居”和“最终目标”的信念分布。

### 6. 论文的主要结论与发现
1.  **叠加态的坍缩**：在免训练和微调制度下，LLM 表现出明显的“叠加坍缩”。模型在处理软 token 时，在前几层就会迅速将其锁定为某个单一的离散解释，其内部熵曲线与离散 CoT 几乎重合。
2.  **快捷方式学习**：微调后的 Coconut 模型实际上学会了“作弊”。探测发现，模型在第一个隐式步骤就已经识别出了最终答案，后续的隐式 token 只是在“复制”答案，而非进行多步推理。
3.  **从头训练的例外**：只有从零开始训练的小型模型展现出了真正的叠加迹象（能够同时维持多个可能的推理路径）。
4.  **两大核心障碍**：
    *   **预训练偏见**：基于“预测下一个 token”的任务使模型在最后几层具有强烈的“承诺”倾向（熵骤降）。
    *   **容量陷阱**：模型容量越大，越倾向于寻找直接从问题到答案的“快捷方式”，而不是利用隐式空间进行复杂的并行搜索。

### 7. 优点（亮点）
*   **批判性视角**：挑战了当前隐式推理研究中关于“叠加”优势的流行假设，揭示了其背后的“幻觉”。
*   **分析工具深入**：不仅看结果，更通过 Logit Lens 和信念演化图（Belief Evolution）展示了模型推理的动态过程。
*   **统一解释**：提出了预训练目标和模型容量是导致叠加失效的两个互补原因，为未来设计隐式推理模型提供了原则性指导。

### 8. 不足与局限
*   **任务类型限制**：主要集中在数学和合成逻辑任务，对于更具创造性或开放性的生成任务，隐式叠加是否有效尚不明确。
*   **方法覆盖面**：虽然分析了 Soft Thinking 和 Coconut，但尚未覆盖所有新兴的隐式推理变体（如基于强化学习训练的连续 CoT）。
