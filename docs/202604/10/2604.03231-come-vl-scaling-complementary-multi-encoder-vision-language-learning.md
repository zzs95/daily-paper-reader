### 1. 论文的核心问题与整体含义
*   **研究动机**：当前的视觉-语言模型（VLM）大多依赖单一的对比学习视觉编码器（如 CLIP 或 SigLIP）。虽然这类编码器擅长全局语义对齐，但在处理需要细粒度空间信息的任务（如物体计数、精准定位、指向性任务）时表现不佳。
*   **核心问题**：如何有效地融合互补的视觉编码器（对比学习型与自监督学习型），在不显著增加大语言模型（LLM）计算负担的前提下，提升模型的视觉理解与定位（Grounding）能力。

### 2. 论文提出的方法论
CoME-VL 框架引入了三个关键技术模块：
*   **熵引导的层选择（Entropy-guided Layer Selection）**：通过分析不同编码器各层的 Token 熵值发现，SigLIP2 在深层仍保持高熵（捕捉多样语义），而 DINOv3 的熵随深度增加迅速下降（聚焦于特定空间结构）。据此选择 SigLIP2 的全层与 DINOv3 的中深层（10-23层）进行融合。
*   **正交正则化多层混合（Orthogonality-Regularized Mixing, OL）**：为了减少相邻层之间的特征冗余，在聚合前对各层输出应用轻量级的正交投影。通过 Cayley 变换等数学手段约束投影矩阵的正交性，确保融合后的特征能占据互补的子空间。
*   **RoPE 增强的交叉注意力对齐（RoPE-enhanced Cross-Attention）**：针对不同编码器 Token 网格分辨率不一致的问题，使用交叉注意力机制（以 SigLIP 为 Query，DINO 为 Key/Value）进行对齐。引入旋转位置嵌入（RoPE）来注入相对空间结构，从而在不增加 LLM 输入 Token 数量的情况下实现高效融合。

*   **层级特性**：DINO 的后期层对定位至关重要，而 SigLIP 的多层聚合能提供更丰富的语义背景。
