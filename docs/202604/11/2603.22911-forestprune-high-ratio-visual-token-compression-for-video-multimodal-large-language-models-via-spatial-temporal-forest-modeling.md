### 1. 核心问题与研究动机
*   **核心问题**：视频多模态大语言模型（Video MLLMs）在处理长视频或多帧输入时，视觉 Token 数量呈爆炸式增长，导致极高的计算延迟和内存开销。
*   **研究动机**：现有的 Token 压缩方法（如剪枝或合并）多源于图像任务，在处理视频时往往只关注单帧内的重要性，忽略了帧间的时间连续性和空间冗余。这导致在高压缩比（如 90% 以上）下，模型性能会剧烈下降，且保留的 Token 在相邻帧间存在大量重复。

### 2. 核心方法论：ForestPrune
ForestPrune 是一种**无需训练（Training-free）**的剪枝方法，其核心思想是通过**时空森林建模**来评估 Token 的全局重要性。
*   **时空森林构建**：
    1.  **候选节点选择**：首先从每帧中初步筛选代表性 Token 作为树节点。
    2.  **建立连接**：基于**语义相似度**（余弦相似度）、**空间约束**（欧氏距离）和**时间顺序**（帧先后）三个维度，将不同帧的相似 Token 连接起来。
    3.  **森林生成**：通过算法将这些连接组织成多棵“树”，形成森林。每一棵树代表视频中一个跨越时空的语义对象或背景区域。
*   **Token 剪枝策略**：
    1.  **树深度评估**：认为深度越大的树（跨帧越多的语义序列）越重要。
    2.  **角色化剪枝**：优先剪掉树的“叶子节点”（末端冗余）和“尾部节点”，保留“根节点”和“主干节点”。
    3.  **全局决策**：在整个视频范围内根据预设的压缩预算（Budget）进行全局最优选择，而非简单的每帧等比例剪枝。
