*   **自训练机制（Self-training）**：
    1.  首先使用少量标注数据训练一个初始模型。
    2.  利用该模型对大量未标注数据进行预测，生成“伪标签”（Pseudo-labels）。
    3.  筛选高置信度的伪标签数据加入训练集，循环往复，实现模型的自我进化。
