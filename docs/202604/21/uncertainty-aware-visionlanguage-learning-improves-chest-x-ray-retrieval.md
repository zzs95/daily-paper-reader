### 2. 方法论：MedProbCLIP
*   **核心思想**：将图像和文本不再映射为单一向量，而是映射为**多变量高斯分布**（Multivariate Gaussian Distributions）。
*   **关键技术细节**：
    *   **概率嵌入**：编码器（Encoder）不仅输出均值向量 $\mu$（代表特征的核心语义），还输出方差向量 $\sigma$（代表该特征的不确定性）。
    *   **分布表示**：对于给定的图像 $i$ 和文本 $t$，其表征分别为 $p(z_i|x_i) = \mathcal{N}(\mu_i, \Sigma_i)$ 和 $p(z_t|x_t) = \mathcal{N}(\mu_t, \Sigma_t)$。
    *   **损失函数**：采用概率对比损失（Probabilistic Contrastive Loss）。不同于计算点积，该方法计算两个分布之间的重叠程度（如使用 Expected Likelihood Kernel 或 KL 散度），使相关样本的分布尽可能重合，不相关的分布尽可能远离。
    *   **不确定性量化**：通过学习到的方差 $\sigma$ 直接衡量模型对特定样本的信心。
