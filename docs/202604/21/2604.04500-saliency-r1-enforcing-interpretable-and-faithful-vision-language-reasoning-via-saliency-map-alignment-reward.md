该论文针对视觉-语言模型（VLMs）在推理过程中的**不忠实性（Unfaithfulness）**和**低解释性**问题展开研究。
*   **研究动机**：尽管当前的 VLM（如 DeepSeek-R1 风格的模型）能生成详细的思维链（CoT），但研究发现这些“思考过程”往往与视觉证据脱节。模型可能过度依赖文本先验而忽视图像，或者生成的推理逻辑与最终答案不一致（即幻觉）。
*   **核心目标**：通过一种可量化的方式，强制模型的推理过程（Thinking Tokens）必须真正“看到”并“利用”图像中的关键区域，从而实现推理逻辑与视觉感知的对齐。

### 2. 论文提出的方法论
Saliency-R1 框架结合了高效的显著性检测技术与强化学习对齐策略：
*   **基于 Logits 分解的显著图生成**：
    *   利用 Transformer 的残差结构，将预测 Token 的 Logits 直接分解为各个输入 Token（包括视觉 Token）的贡献。
    *   **关键技术**：该方法不需要额外的反向传播或多次前向推理，仅通过矩阵运算即可实时提取视觉贡献度，计算开销极低。
*   **以“思考 Token”为瓶颈的注意力回溯（Attention Rollout）**：
    *   为了追踪信息流，论文设计了一种机制：衡量视觉信息如何通过中间的“思考 Token”传递到最终的“答案 Token”。
    *   这种方法能识别出哪些图像区域真正驱动了整个推理链条。
*   **显著性对齐奖励（Saliency-map Alignment Reward）**：
    *   计算生成的显著图与人工标注的关键区域（Bounding Box）之间的重合比例（Recall）。
    *   **优化算法**：采用 **GRPO（群组相对策略优化）**。奖励函数由三部分组成：准确率奖励（LLM 评判）、格式奖励（是否符合 `<think>` 标签）和显著性对齐奖励。
