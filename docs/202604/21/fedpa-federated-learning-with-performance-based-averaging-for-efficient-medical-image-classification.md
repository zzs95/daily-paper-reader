### 论文总结：FedPA — 基于性能加权平均的高效医学图像分类联邦学习

#### 1. 核心问题与整体含义（研究动机和背景）
在医学影像领域，由于隐私保护和数据安全法规（如GDPR、HIPAA），各医疗机构难以直接共享原始数据。**联邦学习（Federated Learning, FL）**作为一种去中心化的训练范式，允许在不交换数据的前提下协作训练模型。
然而，医学数据在不同机构间存在严重的**非独立同分布（non-IID）**问题（如设备差异、病患群体差异、标注标准不一）。传统的联邦平均算法（FedAvg）主要依据各客户端的数据量进行加权，忽略了客户端更新质量的差异。如果某个客户端的数据质量差或分布极度偏离，其贡献可能会损害全局模型的性能。本研究旨在通过引入**性能评价机制**，动态调整客户端权重，以提升模型在异构医学数据下的收敛速度和分类精度。

#### 2. 论文提出的方法论：FedPA
**核心思想**：将全局模型的聚合逻辑从“基于样本量加权”转变为“基于模型性能表现加权”。

*   **关键技术细节与流程**：
    1.  **本地训练**：各参与方（客户端）下载全局模型，利用本地医学影像数据进行多轮迭代训练。
    2.  **性能评估**：训练完成后，客户端在本地验证集上对更新后的模型进行评估，计算性能指标（如准确率 Accuracy 或 F1-score）。
    3.  **参数上传**：客户端将模型更新（权重梯度）连同该轮次的**性能得分**一并发送至中央服务器。
    4.  **性能加权聚合（Performance-Based Averaging）**：服务器不再单纯按数据量分配权重，而是根据各客户端反馈的性能得分计算权重系数。表现越好的模型，其在全局模型更新中的占比越高。
    5.  **全局更新**：服务器生成新的全局模型并分发，进入下一轮循环。
