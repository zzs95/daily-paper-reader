### 2. 方法论：MedSegAgent 系统架构
MedSegAgent 采用了一种**多智能体协作框架**，通过“由粗到精”的过滤机制实现指令到分割结果的转化。系统包含五个核心智能体：
1.  **查询解析智能体 (Query Parsing Agent)**：利用大语言模型（LLM）解析用户的自然语言请求，提取出模态、解剖部位和目标标签等结构化意图。
2.  **模态过滤智能体 (Modality Filtering Agent)**：根据解析出的意图，从模型库中筛选出匹配的影像模态（如CT、PET、超声等）。
3.  **解剖过滤智能体 (Anatomical Filtering Agent)**：在选定模态的基础上，进一步筛选覆盖目标解剖区域（如腹部、头部）的候选数据集和模型。
4.  **标签选择智能体 (Label Selection Agent)**：在最精细的层面，匹配具体的分割标签值（如肝脏、肿瘤、血管等）。
5.  **执行智能体 (Execution Agent)**：负责调用底层预训练模型进行推理，并对多个可能的结果进行整合与后处理。
