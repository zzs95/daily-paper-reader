### 1. 核心问题与整体含义（研究动机和背景）
现有的视觉-语言模型（VLM）大多依赖单一的视觉编码器（如 CLIP 或 SigLIP），这类编码器通过对比学习获得强大的全局语义对齐能力，但在**细粒度视觉定位、计数和空间关系理解**方面表现不足。此外，随着语言模型（LLM）规模的扩大，单一视觉编码器往往成为系统性能的瓶颈。本文旨在探索如何通过融合**对比学习编码器（SigLIP2）**与**自监督学习编码器（DINOv3）**的互补优势，提升 VLM 在理解与定位（Grounding）任务上的综合表现。

### 2. 方法论：核心思想与关键技术
CoME-VL 提出了一个模块化的多编码器融合框架，核心技术包括：
*   **互补双编码器架构**：集成 SigLIP2（提供强语义对齐）和 DINOv3（提供丰富的几何与空间特征）。
*   **熵引导的层选择（Entropy-guided Selection）**：通过分析不同层 token 的空间熵，发现 SigLIP2 的语义特征分布在全层，而 DINOv3 的空间定位特征在深层（10-23层）更集中，据此选择最优层范围。
*   **正交正则化混合（Orthogonality-Regularized Mixing）**：为了减少多层特征间的冗余，引入“正交层（OL）”对各层输出进行投影，通过 skew-symmetric 矩阵参数化确保投影矩阵的近等距性，使融合特征更具多样性。
*   **RoPE 增强的交叉注意力对齐**：使用 SigLIP2 token 作为 Query，DINOv3 token 作为 Key/Value，并引入二维旋转位置嵌入（RoPE）进行空间对齐。这种方式比直接拼接 token 更高效，避免了 LLM 上下文长度的剧增。

    *   基准模型：**Molmo-7B**（基于 Qwen2-7B）。

*   **任务类型**：涵盖视觉问答、图表推理、细粒度计数、坐标点预测（Pointing）和边界框检测（Localization）。

### 6. 主要结论与发现
*   **互补性证实**：SigLIP 擅长语义理解，DINO 显著增强了空间定位和计数能力。

### 7. 优点（亮点）

*   **端到端定位**：成功将原本不支持检测的 Molmo 模型扩展为具备精确坐标输出能力的定位模型。

*   **推理开销**：虽然比拼接法快，但相比单编码器模型仍有约 20% 的耗时增加。
