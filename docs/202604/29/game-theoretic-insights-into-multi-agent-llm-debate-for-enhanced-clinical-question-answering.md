*   **整体含义**：本研究不仅在实证上验证了 MAD 的有效性，还首次从博弈论的角度为多智能体协作提供了理论证明，解释了为何以及在何种条件下辩论能优于单模型推理。

### 2. 论文提出的方法论
*   **核心思想**：将临床问答建模为一个协作博弈。多个具有不同“人格”（Personas）的智能体针对同一医学问题进行多轮讨论，通过互相纠错和证据交换来逼近正确答案。
*   **关键技术细节**：
    *   **角色多样性（Role Diversity）**：为不同智能体分配特定角色（如：侧重临床指南的专家、侧重罕见病的专家等），以鼓励不同的推理路径。
    *   **博弈论形式化**：
        *   **定理1（多数投票改进）**：基于孔多塞陪审团定理，证明当智能体独立且准确率 > 0.5 时，多数投票能显著提升整体准确率。
        *   **定理2（加权聚合优化）**：证明在智能体置信度经过校准的前提下，按置信度对数赔率（Log-odds）加权是贝叶斯最优聚合策略。
    *   **早停机制（Early Stopping）**：引入共识判断，若智能体在预设的最大轮次（T）前达成一致，则提前结束辩论。
*   **算法流程**：
    1.  **初始化**：所有智能体接收问题并生成初始答案及理由。
    2.  **辩论循环**：智能体阅读彼此的理由，更新自己的观点。
    3.  **早停检查**：若达成多数共识，则停止。
    4.  **最终聚合**：通过多数投票或 LLM 法官（Judge）给出最终答案。
