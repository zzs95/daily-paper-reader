该研究针对胸部放射影像（CXR）自动诊断中的关键挑战：**如何将视觉放射学特征与复杂的临床文本描述有效对齐**。传统的深度学习模型高度依赖大规模标注数据，且难以识别训练集中未出现的疾病（零样本场景）。论文旨在通过视觉-语言模型（VLM）和神经记忆机制，提升模型对已知及未见肺部疾病的检测准确性、泛化能力和临床可解释性。

### 2. 论文提出的方法论
论文提出了 **DVLM（具有神经记忆的双头视觉-语言模型）** 框架，其核心组成部分包括：
*   **双头架构**：并行结合了**对比学习分支**（用于视觉-文本对齐）和**监督学习分支**（用于利用已知标签强化特征提取）。
*   **编码器选择**：视觉端采用 **Vision Transformer (ViT-B/16)** 捕捉长程依赖；文本端采用 **ClinicalBERT** 处理放射学报告中的专业术语。
*   **神经记忆模块 (Neural Memory Module)**：这是核心创新点，用于在训练过程中存储和检索疾病相关的关键模式，从而在面对未见过的病理（Zero-Shot）时提供类比参考。
*   **概率校准**：引入**温度缩放 (Temperature Scaling)** 技术，将模型预测的置信度与实际准确率对齐，减少医疗诊断中的过度自信问题。
