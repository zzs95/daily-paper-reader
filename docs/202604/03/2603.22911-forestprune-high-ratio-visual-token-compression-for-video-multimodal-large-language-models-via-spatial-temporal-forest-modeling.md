### 1. 核心问题与研究动机
*   **核心问题**：视频多模态大语言模型（Video MLLMs）在处理长视频时，由于帧数多、分辨率高，会产生海量的视觉 Token，导致极高的计算延迟和内存开销。
*   **研究动机**：现有的 Token 压缩方法（如剪枝或合并）多源于图像任务，缺乏对视频特有的**时空连续性**和**全局冗余**的建模。在极高压缩比（如 90%）下，现有方法往往因无法识别跨帧的重复信息而导致性能剧降。

### 2. 核心方法论：ForestPrune
ForestPrune 是一种**无需训练（Training-free）**的即插即用方法，其核心思想是将视频 Token 建模为“时空森林”。
*   **候选节点选择**：首先从每帧中初步筛选代表性 Token 作为树节点。
*   **时空森林构建**：
    *   **约束机制**：基于**语义相似度**（余弦相似度）、**空间对齐**（坐标距离）和**时间顺序**（帧先后）建立节点间的连接。
    *   **树结构生成**：通过计算入度确定根节点，并根据时空得分将子节点分配给最匹配的树。
*   **全局剪枝策略**：
    *   **重要性评估**：认为树越深（跨帧越长）的节点越重要，根节点和主干节点比叶子节点更重要。
    *   **预算化剔除**：根据预设的压缩比例，优先剪掉森林中的叶子节点和尾部节点，从而在全局范围内保留最具代表性的视觉特征。

### 6. 主要结论与发现

### 7. 优点与亮点
*   **时空建模新颖**：将视频冗余问题转化为森林剪枝问题，直观且有效地利用了视频的帧间相关性。
*   **无需训练**：作为一种插件式方法，不需要昂贵的重训练成本，易于部署到现有的 MLLMs 中。
*   **全局最优**：相比于逐帧剪枝，ForestPrune 实现了跨帧的全局重要性评估，有效解决了“相邻帧保留相似 Token”的冗余问题。

### 8. 不足与局限
*   **超参数敏感性**：虽然消融实验显示其具有一定鲁棒性，但语义和空间阈值（$\tau_s, \tau_p$）仍需针对不同模型或视频类型进行微调以达到最优效果。
*   **极端场景限制**：在镜头切换极快或内容极度不连续的视频中，时空树的构建可能变得破碎，导致压缩效果退化。
*   **依赖初始筛选**：森林的质量部分取决于第一步候选节点的筛选质量，若初始筛选丢失了关键小物体，后续森林建模也难以找回。
