### 1. 核心问题与研究背景
*   **核心问题**：放射科报告生成（RRG）旨在根据医学图像自动生成诊断文本。现有的方法通常将报告视为扁平的单词序列，忽略了报告内部的层级结构（如段落、句子、单词）以及这些结构与视觉特征之间的细粒度对应关系。
*   **研究背景**：放射科医生工作量巨大且易产生人为疲劳错误。虽然现有的视觉-语言模型在通用领域表现良好，但在医学领域，由于医学术语的专业性和报告的长文本特性，实现精确的跨模态对齐（Cross-modal Alignment）仍然是一个巨大的挑战。

### 2. 论文提出的方法论
RIHA（报告-图像层级对齐 Transformer）框架的核心思想是通过多层次的对齐机制来捕捉视觉特征与临床叙述之间的细微语义关联。
*   **视觉特征金字塔（VFP）**：用于提取多尺度的视觉特征，从全局解剖结构到局部的病灶细节。
*   **文本特征金字塔（TFP）**：将报告表示为多粒度的文本结构，分别对应单词、句子和段落级别。
*   **跨模态层级对齐（CHA）模块**：这是核心组件，利用**最优传输（Optimal Transport, OT）**算法，在段落、句子和单词三个层级上分别计算视觉与文本特征的对齐成本，从而实现更精准的跨模态映射。
*   **相对位置编码（RPE）**：在解码器中引入 RPE，以建模 Token 之间的空间和语义关系，增强生成文本时的逻辑连贯性。
