### 1. 论文的核心问题与整体含义
*   **核心问题：** 视觉-语言模型（VLM）在进行有监督微调（SFT）以提升感知能力（如OCR、图表理解）时，往往会导致推理能力（如数学、科学推理）的显著下降。这种现象被称为**“推理税”（Reasoning Tax）**。
*   **研究动机：** 作者假设预训练模型在不同深度形成了特征层级（浅层编码感知，深层编码抽象推理）。统一的微调破坏了这种层级访问。研究旨在探索是否可以通过恢复对跨深度表示的访问来缓解推理能力的退化。

### 2. 论文提出的方法论：IADA
*   **核心思想：** 提出**输入自适应深度聚合（Input-Adaptive Depth Aggregation, IADA）**。这是一种轻量级模块，插入在Transformer块之间，允许模型根据当前输入动态地从之前的层中检索信息。
*   **关键技术细节：**
    *   **输入自适应查询生成：** 对当前隐藏状态进行平均池化，通过一个**低秩瓶颈层（Low-rank Bottleneck）**生成查询向量（Query）。
    *   **模态感知路由：** 针对视觉和文本标记（Tokens）分别维护独立的路由流，使用二进制掩码分离模态，确保检索过程具有模态特异性。
    *   **跨深度检索：** 使用生成的查询向量，通过交叉注意力机制（Cross-Attention）从之前所有块边界缓存的隐藏状态中检索信息。
    *   **低秩门控注入：** 检索到的残差通过一个可学习的门控机制（初始化为接近0）注入回当前隐藏状态，确保训练初期的稳定性。
*   **算法流程：** 缓存历史状态 -> 池化当前状态 -> 低秩投影生成查询 -> 跨深度注意力检索 -> 门控残差相加。

### 6. 论文的主要结论与发现
*   **推理税的确认：** 仅使用 LoRA 微调使 Qwen3-VL 的平均推理得分下降了5.1分。
*   **跨深度访问的重要性：** 即使是简单的固定查询聚合也能恢复大部分推理能力，证明了“访问丢失”是推理税的关键因素。

*   **低秩效应：** IADA 在低秩 LoRA 设置下增益最明显，表明它与传统的层内适配（PEFT）具有结构互补性。
*   **输入自适应是核心：** 相比模态分离，输入自适应的查询生成对性能提升的贡献更大。
