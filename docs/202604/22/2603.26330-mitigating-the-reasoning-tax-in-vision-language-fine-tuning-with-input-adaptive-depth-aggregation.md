### 1. 论文的核心问题与整体含义
*   **研究动机**：视觉-语言模型（VLM）在进行有监督微调（SFT）时，往往会出现“推理税”（Reasoning Tax）现象——即模型在提升视觉感知能力（如OCR、图像识别）的同时，其逻辑推理、数学和科学分析能力会显著下降。
*   **核心假设**：作者认为这种退化是因为微调破坏了模型对预训练阶段形成的“深度特征层次结构”的访问。浅层通常编码感知模式，深层编码抽象推理。统一的微调可能干扰了这种层次间的有效信息流动。
*   **研究目标**：探索如何通过恢复和优化跨深度（Cross-depth）的信息检索，在不牺牲感知增益的前提下缓解推理能力的损失。

### 2. 论文提出的方法论：IADA
作者提出了**输入自适应深度聚合（Input-Adaptive Depth Aggregation, IADA）**，这是一种轻量级的结构化微调增强机制：
*   **核心思想**：在 Transformer 块的边界插入模块，允许模型根据当前输入动态地从之前的层中检索有用的特征表示。
*   **关键技术细节**：
    1.  **输入自适应查询生成**：通过对当前隐藏状态进行均值池化，并经过一个低秩瓶颈（Low-rank bottleneck，由 $W_{down}$ 和 $W_{up}$ 组成）生成查询向量（Query）。
    2.  **模态感知路由**：针对视觉 Token 和文本 Token 分别维护独立的路由流。虽然骨干网络共享注意力，但 IADA 允许两种模态根据自身需求选择不同的深度特征。
    3.  **跨深度检索**：利用交叉注意力机制，以生成的自适应查询从缓存的先前块边界隐藏状态中检索信息。
    4.  **低秩门控注入**：检索到的残差通过一个初始值为零的门控机制（Sigmoid 激活）注入回当前隐藏状态，确保训练初期的稳定性。
*   **算法流程**：将网络分为 $K$ 个块，在每个块结束时，计算模态特定的查询 -> 从历史块中检索特征 -> 门控融合 -> 更新隐藏状态。

### 6. 论文的主要结论与发现
1.  **证实推理税存在**：仅使用 LoRA 微调会导致推理均分下降 5.1 分。
2.  **跨深度访问是关键**：即使是简单的固定深度聚合也能恢复大部分推理能力，证明了“访问丢失”是推理退化的主因。
3.  **自适应性至关重要**：IADA 相比固定聚合，在推理上进一步提升，尤其在 OCRBench 等需要精细对齐的任务上表现突出。
4.  **低秩高效性**：仅需 0.14M 参数，IADA 就能在推理均分上比 LoRA-only 提升 **9.5 分**，感知均分提升 **3.3 分**。
5.  **互补性**：IADA 与 LoRA 是正交的，它在低秩 LoRA 设置下增益最大，说明它弥补了 PEFT 在跨层信息流建模上的不足。

### 7. 优点（亮点）
*   **视角新颖**：将微调退化问题转化为“结构化信息访问”问题，而非单纯的训练稳定性或数据分布问题。
*   **极其轻量**：以极小的参数代价（<0.2M）实现了显著的性能回升，非常适合端侧或资源受限的微调场景。
*   **即插即用**：作为 PEFT 的补充，不改变原始 Transformer 层的计算逻辑，易于集成到现有框架。

### 8. 不足与局限
*   **模型单一**：实验主要集中在 Qwen3-VL-2B 上，尚未在更大规模（如 7B, 72B）或其他架构（如 InternVL, LLaVA）上验证泛化性。
*   **数据局限**：仅使用了一个指令微调数据集，不同数据分布对“推理税”的影响程度可能不同。
*   **模态掩码依赖**：依赖于固定的模态掩码来区分视觉和文本，在深度融合的层中，这种区分可能变得模糊。
*   **生成质量评估缺失**：主要关注 Benchmark 分数，缺乏对模型长文本生成质量或自由对话连贯性的定性分析。
