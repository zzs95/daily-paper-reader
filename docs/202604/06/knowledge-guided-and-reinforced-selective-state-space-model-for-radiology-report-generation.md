### 1. 核心问题与整体含义（研究动机和背景）
*   **研究背景**：放射学报告生成（RRG）是医疗人工智能的重要领域，旨在根据医学影像自动生成描述性文本。
*   **核心问题**：
    1.  **模型黑盒化**：现有的序列模型（如 Transformer）缺乏透明度，难以解释其生成逻辑。
    2.  **奖励机制单一**：传统的强化学习奖励函数（如 BLEU、ROUGE）侧重于语言重合度，忽略了医学事实的准确性。
    3.  **临床一致性不足**：生成的报告可能在语法上通顺，但在临床诊断上存在错误或遗漏关键医学知识。

### 2. 论文提出的方法论
该研究提出了一种结合**选择性状态空间模型（Selective State Space Model, SSM）**、**医学知识引导**与**强化学习优化**的框架：
*   **核心架构（SSM/Mamba）**：采用选择性状态空间模型（推测为 Mamba 架构）作为骨干网络。相比 Transformer，SSM 在处理长序列时具有线性复杂度，且通过选择性机制能更好地捕捉医学影像与文本之间的长程依赖。
*   **知识引导机制**：引入显性医学知识（如医学知识图谱或本体库），在编码或解码阶段引导模型关注特定的解剖结构和病理特征，减少“幻觉”生成。
*   **强化学习优化（RL）**：设计了改进的奖励函数，不仅包含语言质量指标，还整合了临床准确性奖励（如基于疾病标签的一致性得分），通过策略梯度算法优化模型，使其生成更符合临床事实的报告。
