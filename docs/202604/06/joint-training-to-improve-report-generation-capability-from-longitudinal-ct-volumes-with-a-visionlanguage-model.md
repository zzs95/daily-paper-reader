*   **核心问题**：放射科医生在诊断时通常需要对比患者当前的CT影像与历史影像（纵向分析），以观察病灶变化。然而，手动编写此类对比报告耗时耗力。
*   **研究动机**：虽然视觉语言模型（VLM）在单幅图像描述上取得了进展，但在处理具有时间跨度的3D CT卷（Volumes）并生成准确的对比性描述方面仍面临挑战。
*   **背景**：医学影像数据（尤其是带标注的纵向数据）相对稀缺，如何有效利用有限的数据提升模型对病变演变的理解能力是当前的研究重点。

*   **核心思想**：采用**联合训练（Joint Training）**策略。研究者认为，仅使用纵向数据训练模型可能导致过拟合或特征提取不充分，通过同时引入单时期（Single-time-point）数据进行联合训练，可以增强模型的基础视觉表征能力。

    *   **模型架构**：基于视觉语言模型（VLM），通常包含一个3D图像编码器（处理CT卷）和一个文本解码器（生成日语文本）。
    *   **输入处理**：模型同时接收“当前CT”和“既往CT”作为输入，通过注意力机制或特征融合模块捕捉两者之间的差异。
    *   **训练策略**：对比了三种模式：1) 仅使用纵向数据训练；2) 先在单时期数据上预训练再在纵向数据上微调；3) **联合训练**（同时输入单时期和纵向数据，共享模型参数）。
