### 1. 核心问题与整体含义（研究动机和背景）
论文关注视觉-语言模型（VLM）在后训练阶段面临的一个普遍问题——**“推理税”（Reasoning Tax）**。即在通过有监督微调（SFT）提升模型视觉感知能力（如OCR、图表理解）时，往往会导致其逻辑推理能力（如数学、科学推理）显著下降。

作者假设这种退化是因为微调破坏了模型在预训练中形成的“深度特征层级”：浅层编码感知模式，深层编码抽象推理。微调使模型难以有效访问这些分布在不同深度的推理相关表示。因此，研究的核心目标是**通过恢复和优化跨深度的信息访问，在不牺牲感知增益的前提下挽救推理能力**。

### 2. 论文提出的方法论：IADA
作者提出了**输入自适应深度聚合（Input-Adaptive Depth Aggregation, IADA）**，这是一种轻量级的结构化插件，其核心思想是在 Transformer 块之间建立显式的跨层信息检索路径。

*   **核心流程**：
    1.  **输入自适应查询生成**：不使用固定查询，而是对当前层的隐藏状态进行均值池化，通过一个**低秩瓶颈（Low-rank bottleneck）**结构（$W_{down}$ 和 $W_{up}$）生成与当前输入内容相关的查询向量（Query）。
    2.  **模态感知路由**：考虑到视觉和文本 token 的处理模式不同，IADA 为两种模态维护独立的路由流。它利用模态掩码分离特征，分别进行跨深度检索。
    3.  **跨深度检索与聚合**：利用生成的查询向量，通过交叉注意力机制（Cross-Attention）从之前所有块边界缓存的隐藏状态中检索有用信息。
    4.  **低秩门控注入**：将检索到的残差信息通过一个初始值为零的门控机制（Sigmoid Gating）注入回当前隐藏状态，确保训练初期的稳定性。
*   **与 PEFT 的关系**：IADA 与 LoRA 等方法正交，LoRA 修改层内变换，而 IADA 优化层间路由。

实验设计非常系统且充分：

### 6. 主要结论与发现
*   **推理税确实存在**：仅使用 LoRA 微调使推理均分下降了 5.1 分。
*   **跨深度访问是关键**：即使是简单的固定深度聚合也能恢复大部分推理能力，证明了作者关于“特征层级访问受阻”的假设。
*   **IADA 表现卓越**：在 Qwen3-VL-2B 上，IADA 相比 LoRA-only 将推理均分提升了 **9.5 分**，感知均分提升了 **3.3 分**。
*   **低秩高效性**：Rank 为 16 的瓶颈效果优于全秩，说明深度路由是一个低维决策问题，且低秩具有正则化作用。
*   **容量敏感性**：IADA 在低秩 LoRA（如 Rank 16）下增益最显著，说明它在参数受限的场景下是极佳的补充。

### 7. 优点
*   **即插即用且轻量**：仅需 0.14M 参数即可获得显著提升，推理成本增加极小。
*   **视角独特**：跳出了传统的“层内优化”思路，从“层间信息流”的角度解决微调退化问题。
*   **模态感知设计**：针对 VLM 的多模态特性，设计了自适应的模态分离路由，符合直觉且实验证明有效。

### 8. 不足与局限
*   **模型通用性验证有限**：实验主要集中在 Qwen3-VL-2B 这一种架构上，尚未在更大规模（如 72B）或其他模型族（如 Llama-VIL）上验证。
*   **模态掩码依赖**：依赖于固定的模态掩码，在视觉与文本特征高度融合的极深层，这种分离可能不再理想。
*   **评估维度**：主要关注 Benchmark 分数，未评估深度聚合对模型生成幻觉、长文本生成质量或对话连贯性的具体影响。
