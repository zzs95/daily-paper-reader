### 1. 论文的核心问题与整体含义
*   **研究动机**：尽管视觉-语言模型（VLMs）表现优异，但其可信度存疑。模型往往过度依赖文本先验（Textual Cues）而忽视视觉证据，导致产生“幻觉”或不合逻辑的推理。
*   **核心问题**：现有的思维链（CoT）推理过程往往与模型实际关注的视觉区域不一致（即“不忠实”）。论文旨在通过一种可解释的机制，强制模型的推理逻辑与图像中的关键视觉区域对齐，确保模型是“看着图说话”而非“盲目猜测”。

### 2. 论文提出的方法论
Saliency-R1 框架结合了高效的显著性图生成技术与强化学习对齐策略：
*   **逻辑值分解（Logits Decomposition）**：提出一种无需额外前向或后向传播的显著性图生成方法。通过将 Transformer 最后一层的输出分解为各个输入 Token 的直接贡献，提取视觉 Token 对预测 Token 逻辑值的贡献度。
*   **注意力滚动（Attention Rollout）瓶颈机制**：为了追踪信息流，该方法将“思考（Thinking）”Token 视为瓶颈。利用注意力滚动技术计算从视觉 Token 到思考 Token、再到答案 Token 的传递贡献，生成反映整个推理路径的全局显著性图。
*   **显著性对齐奖励（Saliency-map Alignment Reward）**：计算生成的显著性图与人类标注的边界框（Bounding Box）之间的重合比例（Recall）。
*   **GRPO 优化**：采用组相对策略优化（Group Relative Policy Optimization），将上述对齐得分作为奖励信号之一，连同准确率奖励和格式奖励，共同引导模型在推理时聚焦于任务相关的视觉区域。
