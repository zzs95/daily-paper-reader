### 1. 核心问题与研究动机
*   **核心问题**：当前的多模态大语言模型（MLLM）在生成文本时，每一解码步骤都会对所有视觉 Token 进行全量关注（Dense Attention）。这与人类视觉感知截然不同——人类在描述场景时会动态地将目光投向（Fixate）相关区域。
*   **研究动机**：全量关注会导致计算开销巨大（尤其在长视频中）且注意力分散。现有的优化方法（如 Token 压缩或基于问题的 KV Cache 剔除）多为一次性处理，无法根据生成过程中的语义变化动态调整视觉焦点。

### 2. 方法论：Gaze Attention 机制
该论文提出了一种模拟人类注视行为的选择性注意力机制，核心组件包括：
*   **注视区域与描述符（Gaze Regions & Descriptors）**：将视觉 KV Cache 在空间/时间上划分为 rh × rw × rt 的紧凑区域。每个区域通过对其中 Key 向量进行均值池化，生成一个轻量级的描述符 $d_{ug}$。
*   **动态视觉路由（Visual Routing）**：在每个解码步骤 $j$，当前的 Query $q_j$ 与所有区域描述符计算相似度，并选择 Top-K 个最相关的区域。只有被选中的区域内的 KV 对才参与该步的注意力计算。
*   **可学习上下文 Token（Learnable Context Tokens）**：为了弥补局部注意力导致的全局信息缺失，模型在每个图像/帧中加入少量（如 4 个）可学习的上下文 Token。这些 Token 始终可见，提供全局视觉背景。
*   **渐进式训练策略（Progressive TopK Schedule）**：为解决稀疏路由训练不稳定的问题，训练初期采用全量关注，随后逐渐减少 K 值，引导模型学习如何精准路由。

### 6. 主要结论与发现

### 7. 优点与亮点

*   **固定区域划分**：目前采用固定的网格划分区域，可能无法完美契合不规则形状的物体边界。
