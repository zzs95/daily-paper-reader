## Abstract

Automated medical report generation plays an important role in clinical medicine, but the current automated medical report generation suffers from some problems: 1) lack of interaction of visual features; 2) insufficient cross-modal interaction. To alleviate these problems, this paper proposes a cross-modal fusion network based on association alignment matrix for automated medical report generation (AAMN). Specifically, we design a Visual Feature Alignment Module (VFAM) to achieve feature alignment by computing the association alignment matrix of two visual features extracted from the front image and side image. Furthermore, we design an effective Cross-modal Feature Fusion Module (CFFM) to enhance the interactive alignment of textual and visual features. We conduct a series of experiments and analysis on two public real datasets, IU X-RAY and MIMIC-CXR, and the final results show that AAMN can better align the visual features of images from different orientations, reaching the current state-of-the-art performance. The code is published at [https://github.com/YJangoLin/AAMN](https://github.com/YJangoLin/AAMN).
