### 1. 核心问题与整体含义（研究动机和背景）
*   **核心问题**：尽管大语言模型在医学考试（如 USMLE）中表现出色，但在高风险的临床应用中，单模型推理仍存在不可靠、校准性差（容易一本正经地胡说八道）的问题。
*   **研究背景**：为了提高答案的鲁棒性，研究者引入了多智能体辩论（Multi-Agent Debate, MAD）机制，让多个模型相互质疑、补充并汇总意见。本研究旨在通过**博弈论**为这种协作模式提供理论支撑，并验证其在医疗场景下的实际效果。

### 2. 方法论
*   **核心思想**：将多智能体辩论建模为一个博弈过程，通过迭代批判和共识机制来逼近最优解。
*   **关键技术细节**：
    *   **博弈论形式化**：论文证明了在模型校准良好的前提下，**多数投票制（Majority Voting）**能显著提升准确率；而**置信度加权聚合（Confidence-Weighted Aggregation）**（由一个 LLM 担任裁判）在理论上是帕累托最优的。
    *   **MAD 流程**：多个智能体独立生成初始答案 -> 互相交换推理过程并进行批判 -> 根据反馈更新自身答案 -> 最终由裁判模型或投票机制汇总。
    *   **早停机制（Early Stopping）**：为了平衡性能与效率，引入了共识检测，一旦智能体达成一致即停止辩论。
