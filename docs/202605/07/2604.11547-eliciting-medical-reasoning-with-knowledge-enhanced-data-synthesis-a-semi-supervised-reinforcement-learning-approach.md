这篇论文介绍了一种名为 **MedSSR** 的框架，旨在通过知识增强的数据合成和半监督强化学习（RL）来提升大语言模型（LLM）的医学推理能力。以下是对该论文的深度总结：

### 1. 核心问题与研究动机
*   **核心问题**：高质量医学推理数据（尤其是罕见病领域）极度匮乏。
*   **研究背景**：现有的医疗模型开发通常依赖于从昂贵的闭源模型（如 GPT-4）中蒸馏思维链（CoT）推理轨迹。然而，这种方法在罕见病等数据稀缺领域提升有限，且生成长推理链的 API 成本极高。
*   **研究目标**：在不依赖昂贵推理链蒸馏的前提下，高效提升模型在通用医疗及罕见病领域的推理表现。

### 2. 方法论：MedSSR 框架
MedSSR 包含两个核心组件，采用“由内而外”的训练范式：

*   **知识增强的数据合成（Knowledge-enhanced Data Synthesis）**：
    *   利用外部权威医学知识库（PubMed、Wikipedia、教材等）和罕见病列表。
    *   通过设置阈值 $\alpha$ 控制合成数据中罕见病的比例。当随机值小于 $\alpha$ 时，检索相关罕见病知识并引导生成模型（如 GPT-4o）合成复杂的医学问题。
*   **半监督强化学习策略**：
    *   **离线伪标签生成**：利用策略模型自身对合成问题进行多次采样，通过**离线多数投票（Majority Voting）**确定伪标签。这种离线方式有效避免了在线强化学习中常见的“奖励作弊（Reward Hacking）”和模型崩溃。
    *   **两阶段训练流程**：
        1.  **自监督强化学习阶段**：在伪标签合成数据上进行训练，激发模型的内在推理潜力。
        2.  **监督强化学习阶段**：在人工标注的真实数据上进行训练，利用外部准确信号进一步增强模型。
    *   **算法选择**：采用 **GRPO**（Group Relative Policy Optimization）算法，通过组内相对奖励实现稳定优化。
