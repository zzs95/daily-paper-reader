*   **核心问题**：能否利用更普及、更经济的计算机断层扫描血管造影（CTA）影像，结合机器学习技术，实现对斑块水平 IPH 的准确预测？

*   **核心思想**：通过从 CTA 影像中提取多维度的定量特征（斑块成分、几何结构、周围组织环境），训练机器学习分类器来识别是否存在 IPH。
