### 1. 核心问题与研究动机
*   **核心问题**：尽管 LLM 在医学考试（如 USMLE）中表现优异，但在高风险的临床决策中，单模型推理容易出现错误或置信度不校准（Uncalibrated）的问题。
*   **研究动机**：探索通过多个智能体（Agents）相互博弈、批判和协作，是否能像人类专家会诊一样，提高答案的准确性、可靠性和透明度。

### 2. 方法论：MAD 框架与博弈论分析
*   **核心思想**：将多智能体辩论建模为一个**协作博弈（Cooperative Game）**，智能体拥有共同的目标（找到正确答案）。
*   **关键技术细节**：
    *   **角色多样性**：为不同智能体分配不同的人设（Persona），如“遵循指南型”、“大胆诊断型”等，以增加推理路径的多样性。
    *   **辩论流程**：初始轮各智能体给出答案和理由；后续轮次中，智能体根据同伴的论点修正自己的观点。
    *   **早停机制（Early Stopping）**：若智能体在早期轮次达成共识，则提前结束辩论以节省算力。
*   **理论证明**：
    *   **定理 1（多数投票改进）**：基于孔多塞陪审团定理，证明当个体准确率 > 0.5 且错误独立时，多数投票能显著提升整体准确率。
    *   **定理 2（加权聚合最优性）**：证明在智能体置信度校准的前提下，按置信度的对数几率（Log-odds）加权投票是贝叶斯最优的。
    *   **命题 1（效率性）**：证明早停机制能在不损失准确率的情况下大幅降低期望轮次。
