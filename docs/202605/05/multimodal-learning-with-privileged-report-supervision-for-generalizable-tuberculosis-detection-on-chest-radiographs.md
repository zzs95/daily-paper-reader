*   **核心问题**：如何在不增加推理负担（仅使用图像）的前提下，利用临床报告中的丰富语义信息来提升肺结核（TB）检测模型的泛化能力和鲁棒性。
*   **研究背景**：
    *   **数据异质性**：TB 影像数据集通常规模较小且具有高度的设备/人群异质性，导致模型跨中心泛化能力差。
    *   **部署限制**：虽然多模态学习（图像+文本）效果好，但在实际筛查场景中，放射科报告往往缺失或延迟，模型必须能独立处理图像。

*   **核心动机**：引入“特权信息学习”（LUPI）框架，将文本作为训练时的“特权监督”，而在推理时丢弃文本分支，实现“训练多模态，推理单模态”。

### 2. 方法论：核心思想与技术细节
*   **核心思想**：采用图像-文本对齐正则化策略。在训练阶段，通过对比学习和分类损失，强制视觉编码器学习与临床描述一致的特征；推理阶段仅保留视觉分支。
*   **关键技术细节**：
    *   **结构化报告生成**：开发了一个确定性的、保护隐私的规则管道，将原始临床简要笔记转换为标准化的结构化报告（包含年龄段、TB 状态、病变位置及伴随症状）。
    *   **双分支架构**：
        *   **视觉分支**：使用 VGG-11 作为骨干网络，提取 512 维特征并投影至 256 维共享空间。
        *   **文本分支**：使用预训练的 **CXR-BERT**（冻结权重），提取 768 维特征并投影至同一 256 维空间。
    *   **损失函数（$L_{total}$）**：
        1.  **分类损失**：图像和文本分支分别计算交叉熵损失，确保共享分类头具有判别力。
        2.  **余弦相似度损失（$L_{cos}$）**：直接拉近配对的图像-文本嵌入距离。
        3.  **对比损失（$L_{NTX}$）**：采用温度缩放的 NT-Xent 损失，在批次内增强正样本对齐并排斥负样本。
    *   **超参数优化**：通过网格搜索确定对比损失权重（$\lambda$）和温度系数（$\tau$），平衡模态对齐与类别区分。
