这篇论文介绍了 **QPath**，一个专为计算病理学设计的提问感知型（Question-Aware）多分辨率视觉语言大模型框架。以下是对该论文的深度结构化总结：

### 1. 核心问题与研究背景
*   **核心问题**：现有的病理多模态大模型（MLLMs）大多依赖**单分辨率**图像分析。然而，在临床实践中，病理学家需要跨多个尺度（从全局组织架构到微观细胞细节）观察全扫描幻灯片（WSI）才能做出准确诊断。
*   **研究背景**：虽然 CLIP 等模型在分类任务上表现优异，但在需要交互式、细粒度推理的视觉问答（VQA）任务中，单分辨率输入往往会导致关键形态学特征的丢失。

### 2. 方法论
QPath 的核心思想是通过**问题引导**来动态选择图像中最相关的区域，并以多分辨率形式输入模型。
*   **问题感知的图像-文本检索**：
    *   利用预训练的病理基础模型（如 CLIP 类模型）作为编码器。
    *   将图像划分为多个补丁（Patches），计算每个补丁与用户提问（Question）之间的语义相似度分数。
*   **动态多分辨率策略**：
    *   **针对感兴趣区域（ROI）**：采用 AnyRes 策略切分补丁，根据相似度分数选择前 $k$ 个最相关的补丁，与原始图像一同输入 MLLM。
    *   **针对全扫描幻灯片（WSI）**：提出一种平衡“空间覆盖度”与“语义相关性”的采样方法。
        *   **峰值采样（Peak-based）**：在相似度热图上寻找局部极大值，确保采样覆盖不同的解剖区域。
        *   **排名采样（Rank-based）**：选取相似度最高的补丁，捕捉最相关的病理特征。
    *   **分辨率分配**：对选中的补丁分配高、中、低三种不同的分辨率，以模拟病理学家的多倍镜观察习惯。
*   **训练阶段**：基于 Qwen2.5-VL-7B，分为两个阶段：1) 视觉编码器与投影层的对齐训练；2) 全参数的指令微调（包含 80k VQA 样本）。
