### 1. 核心问题与整体含义
*   **研究背景**：医学图像的像素级标注成本极高，弱监督学习（如使用涂鸦/Scribble标注）成为缓解标注负担的关键技术。
*   **核心痛点**：
    1.  **伪标签质量低**：弱监督生成的伪标签通常含有大量噪声，导致模型训练不稳定。
    2.  **SAM利用不充分**：虽然 Segment Anything Model (SAM) 具有强大的泛化能力，但现有研究多关注其微调，而忽略了如何通过其核心的“提示驱动（Prompt-driven）”机制在弱监督下生成高质量伪标签。
*   **研究动机**：探索如何利用图像特征作为参考来增强提示（Prompt），从而引导 SAM 生成更可靠的伪标签，提升分割精度。

### 2. 论文提出的方法论
该方法构建了一个双分支架构（SAM 分支 + 轻量级 U-Net 分支），核心技术包括：
*   **特征参考提示适配器 (FRP-Adapter)**：
    *   **参考特征提取器 (RFE)**：利用通道和空间注意力机制，融合来自 SAM 编码器的语义特征和 U-Net 编码器的结构特征，提取出目标区域的显著信息作为“参考”。
    *   **提示适配器 (PA)**：引入一组**可学习的查询 (Learnable Queries)**。这些查询通过交叉注意力机制与参考特征交互，将图像特征转化为“密集提示（Dense Prompts）”。
*   **多粒度提示融合**：将上述生成的“密集提示”与来自涂鸦标注的“稀疏提示（点/线）”相结合，共同输入 SAM 解码器，生成高精度的伪标签。
*   **双边融合模块 (BFM)**：在编码阶段对齐全局特征（Transformer）与局部特征（CNN），增强双分支之间的一致性。
*   **损失函数**：结合了涂鸦监督损失（Scribble Loss）和基于伪标签的一致性监督损失（Consistency Loss）。
