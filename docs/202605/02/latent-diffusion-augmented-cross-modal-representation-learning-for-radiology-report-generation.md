    *   **病变语义表达不足**：细微病变（如肺结节、心脏肥大）在图像中占比较小、对比度低，传统CNN的降采样过程容易丢失这些微弱信号。
    *   **模态交互障碍**：连续的视觉空间特征与离散的文本语义之间存在巨大的“模态鸿沟”，现有的静态记忆或简单融合机制难以实现精确的语义对齐。
*   **研究背景**：随着医疗影像数据激增，放射科医生面临巨大的报告编写压力，开发能够捕捉细微病理特征并生成高质量报告的AI系统具有重要的临床价值。

### 2. 论文提出的方法论：LDCR 框架
论文提出了一种名为 **LDCR** 的潜在扩散增强跨模态表示学习框架，核心包含两个模块：

    *   **核心思想**：利用扩散模型（Diffusion Model）在潜在空间（Latent Space）进行“退化-重建”学习。

*   **异构信息融合模块 (HIFM)**：
    *   **核心思想**：建立共享语义投影空间，实现视觉与文本的双向交互。
