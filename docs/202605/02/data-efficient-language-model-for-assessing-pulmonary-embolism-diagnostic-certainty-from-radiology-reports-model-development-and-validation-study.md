### 1. 核心问题与整体含义（研究动机和背景）
*   **核心问题**：计算机断层扫描肺血管造影（CTPA）是诊断肺栓塞的标准，但报告中常包含模糊的“对冲语言”（Hedging，如“可能”、“不排除”），导致临床医生理解不一，可能引发误诊或过度治疗。
*   **研究动机**：现有的规则方法难以捕捉复杂的上下文，而大型语言模型（LLM）虽然强大，但在医疗私密环境下部署成本高且需要大量标注数据。研究旨在开发一种在小样本（Few-shot）情况下依然表现优异、无需复杂提示工程（Prompt Engineering）的轻量级模型。

### 2. 论文提出的方法论
*   **核心思想**：采用 **SetFit**（Sentence Transformer Fine-tuning）框架，通过对比学习（Contrastive Learning）在极少量标注数据上微调小型预训练模型。
*   **关键技术细节**：
    *   **第一阶段（对比微调）**：使用 `all-roberta-large-v1` 作为基础模型。通过从相同类别中抽取正样本对、不同类别中抽取负样本对，利用余弦相似度损失函数进行微调，使模型学习到区分诊断确定性的语义嵌入。
    *   **第二阶段（分类头训练）**：在微调后的嵌入空间之上，训练一个简单的分类头（如逻辑回归），预测“明确阳性”、“明确阴性”和“可能（不确定）”三个类别。
    *   **特点**：**无需提示词（Prompt-free）**，直接处理文本，避免了提示词敏感性问题。
