*   **核心问题**：如何通过多个 LLM 智能体之间的相互辩论、批判与协作（Multi-Agent Debate, MAD），并结合博弈论的理论框架，来提高临床问答的准确性、可信度和推理效率。

### 2. 方法论
*   **核心思想**：将多智能体辩论过程形式化为一个博弈论问题，通过数学证明寻找最优的答案聚合策略。

    *   **博弈论形式化**：研究了在不同信息分布下，智能体如何通过交互达到纳什均衡或帕累托最优。
    *   **聚合策略**：
        *   **多数投票（Majority Voting）**：作为基准聚合方式，证明其在智能体独立且准确率大于 0.5 时能提升性能。
        *   **置信度加权聚合**：引入一个“LLM 裁判（Judge）”，根据各智能体的校准程度（Calibration）分配权重，理论证明这是校准状态下的最优策略。
    *   **早停机制（Early Stopping）**：设计了共识检测算法，当智能体在早期回合达成一致时即停止辩论，以优化推理成本。
*   **算法流程**：
    1.  多个智能体独立针对医学问题给出初始答案及理由。
    2.  进入辩论轮次，智能体阅读他人的理由，进行自我修正或反驳。
    3.  通过裁判模型或投票机制整合最终答案。
    4.  应用早停逻辑，若达成共识则提前结束。
