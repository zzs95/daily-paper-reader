### 1. 研究动机与背景
*   **核心问题**：病理诊断本质上是一个多尺度分析过程（从宏观组织结构到微观细胞形态）。然而，现有的病理多模态大语言模型（MLLMs）大多依赖**单分辨率**分析，通常将图像缩放到固定尺寸（如 224x224 或 336x336），这导致了关键细微病理特征的丢失。
*   **研究背景**：计算病理学（CPath）正从传统的自动化分析转向基于视觉问答（VQA）的交互式诊断。为了模仿病理学家的“由粗到精”的观察习惯，需要一种能够根据具体问题动态调整关注分辨率的框架。

### 2. 方法论：QPath 框架
QPath 的核心思想是**问题感知（Question-Aware）的多分辨率融合机制**，分为两个层面：
*   **ROI（感兴趣区域）级别处理**：
    *   **切片与检索**：将 ROI 图像切割成多个高分辨率 Patch。
    *   **问题感知检索**：利用问题文本作为引导，通过计算文本向量与 Patch 视觉向量的相关性，检索出前 $k$ 个最相关的 Patch。
    *   **多尺度融合**：将原始的全局低分辨率视图与检索到的局部高分辨率 Patch 共同输入模型，使模型既能看到整体背景，又能捕捉到回答特定问题所需的关键细节。
*   **WSI（全片图像）级别处理**：
    *   **动态采样策略**：针对超大规模的 WSI，提出了一种平衡空间覆盖范围（Spatial Coverage）与语义相关性（Semantic Relevance）的采样方法，确保在有限的计算预算下实现高效的载玻片级理解。

*   **问题感知的优势**：相比于盲目增加分辨率，根据问题动态检索相关区域不仅节省了计算资源，还减少了无关背景噪声对模型推理的干扰。
*   **泛化能力**：QPath 在未见过的 WSI 任务上表现出极强的零样本迁移能力，证明了其学习到了更本质的病理特征。
