### 1. 论文的核心问题与整体含义
*   **研究背景**：掩码扩散语言模型（MDLM）通过并行预测 Token 实现了比自回归模型更高的解码效率。
*   **核心问题**：当 MDLM 尝试同时生成多个 Token（高并行度）时，生成质量会剧烈下降。
*   **根本原因**：论文指出这源于**标记级训练目标与序列级一致性之间的不匹配**。训练时优化的是每个位置的边际分布，而并行解码时隐含了位置间的条件独立假设，导致生成的 Token 组合在局部合理但在全局逻辑上不一致（例如生成“2 + 2 = 5”）。

### 2. 论文提出的方法论：ME-DLM
*   **核心思想**：提出一种“先粗后精”的框架。首先通过标准的掩码扩散生成一个粗略的完整序列，然后利用**基于编辑的细化（Edit-Based Refinement）**步骤，在全局上下文条件下对序列进行局部修正。
*   **关键技术细节**：
    *   **编辑操作**：模型预测每个位置的编辑动作对 $(c_i, n_i)$。其中 $c_i$ 代表替换（或删除符号 `[DEL]`），$n_i$ 代表在该位置后插入的新 Token。
    *   **确定性监督信号**：利用**编辑距离（Edit Distance）**算法计算生成序列与目标序列之间的最小编辑脚本，将其转化为确定性的训练标签。
    *   **两阶段推理**：
        1.  **掩码扩散阶段**：快速生成初始草稿。
        2.  **编辑细化阶段**：迭代应用预测的编辑操作，直到模型预测“空编辑”或达到最大步数。
*   **算法流程**：训练时通过课程学习（Curriculum Learning）逐渐增加扩散深度，使模型学会修正从轻微到严重的各种结构性错误。
