### 1. 核心问题与整体含义（研究动机和背景）
*   **核心问题**：超声报告自动生成面临三大挑战：
    1.  **图像变异性与操作者依赖性**：超声检查高度依赖医生手法，图像质量不一。
    2.  **语义鸿沟**：视觉特征与复杂的医学文本描述之间存在巨大的理解偏差。
    3.  **数据冗余**：在临床场景中，一个报告通常对应多张图像，但其中包含大量重复或与诊断描述无关的冗余信息，现有模型往往忽略了这一点。
*   **研究背景**：旨在通过引入外部医学知识和自适应机制，减轻临床医生的文书负担，提高报告的临床准确性和逻辑连贯性。

### 2. 论文提出的方法论
RA-DMN 模型由以下四个核心模块组成：
*   **冗余自适应视觉编码器（Redundancy-Adaptive Visual Encoder）**：采用**双重注意力机制（Dual Attention Mechanism）**，从多张输入图像中选择性地融合有效视觉信息，剔除冗余干扰。
*   **双记忆网络（Dual Memory Network）**：
    *   **静态记忆模块**：存储细粒度的临床属性和先验医学知识。
    *   **动态记忆矩阵**：用于记录和演化生成过程中的上下文信息。
*   **记忆对齐模块（Memory Alignment Module）**：将原始视觉特征与基于知识的表示进行桥接，并动态更新上下文记忆，确保视觉与文本语义的一致性。
*   **知识注入解码器（Knowledge-Injected Decoder）**：整合全局诊断知识与动态上下文引导，最终生成临床准确且连贯的文本报告。
