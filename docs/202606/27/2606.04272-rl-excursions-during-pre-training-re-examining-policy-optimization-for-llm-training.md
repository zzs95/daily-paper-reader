本研究的核心在于挑战大语言模型（LLM）训练的“标准范式”——即先进行大规模预训练（PT），再进行有监督微调（SFT），最后才进行强化学习（RL）。

*   **整体含义**：论文证明了 RL 可以在预训练极早期介入，且在保留模型通用能力和扩展解空间分布方面优于传统的 SFT->RL 路径。

### 2. 论文提出的方法论
作者通过从零开始训练模型，并在预训练的中间检查点（Checkpoints）应用不同的训练路径进行对比：
*   **核心思想**：将 RL 视为一种与 NTP（下一 token 预测）并行的目标，而非仅仅是最后的对齐工具。
*   **关键技术路径**：
    *   **Direct RL**：直接在预训练中间检查点 $M_t$ 上应用 RL（使用 GRPO 算法），不经过 SFT。
    *   **Parallel Averaging（并行平均更新）**：这是本文提出的新算法。在每一个训练步，同时从 RL 目标和 SFT 目标计算梯度。关键点在于**独立维护**两套优化器状态（Adam 的一阶/二阶动量），计算出各自的参数更新量 $\Delta_{RL}$ 和 $\Delta_{SFT}$ 后取平均值更新模型。
*   **算法流程**：输入当前参数 $\theta$，在同一快照下分别计算 RL 梯度和 SFT 梯度，通过各自优化器得到更新步长，最后执行 $\theta \leftarrow \theta + \frac{1}{2}(\Delta_{RL} + \Delta_{SFT})$。

### 6. 论文的主要结论与发现
1.  **RL 介入极早**：在预训练仅 4B tokens 时（远早于预训练完成），直接应用 RL 就能显著提升推理能力，效果甚至能匹配完整的 SFT->RL 流程。
2.  **分布扩展 vs. 锐化**：传统的 SFT->RL 会导致“锐化”效应（Pass@1 升，Pass@32 降）；而 **Direct RL 会导致“扩展”效应**，同时提升 Pass@1 和 Pass@k，意味着模型发现了新的推理路径。
3.  **通用能力保留**：SFT 会显著损害模型的非数学通用能力（下降 4-8%），而 **RL 几乎不损害通用能力**。
4.  **并行平均最优**：提出的 Parallel Averaging 算法在 Pass@32 指标上超越了所有单路径方法，且保留了通用能力。
5.  **数据胜过规模**：对于 RL 的效果，增加预训练中的相关领域数据（如数学）比单纯增加模型参数规模更有效。

### 7. 优点
*   **视角独特**：打破了“RL 只能作为最后微调阶段”的思维定式。
*   **发现深刻**：揭示了 SFT 才是导致模型分布收缩和通用能力下降的“元凶”，而非 RL 本身。
*   **实用性强**：提出的并行平均算法简单有效，为未来将 RL 整合进预训练过程提供了可行路径。

### 8. 不足与局限
*   **任务领域局限**：实验主要集中在数学推理（Verifiable Rewards），对于难以自动评估的开放域生成任务（如创意写作、对话）是否适用尚待验证。
*   **模型规模上限**：实验最高只做到了 4B 参数，虽然具有代表性，但在千亿级参数（100B+）模型上的表现可能存在规模效应带来的差异。
