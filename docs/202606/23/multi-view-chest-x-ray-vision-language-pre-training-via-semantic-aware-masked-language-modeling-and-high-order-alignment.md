### 1. 核心问题与研究背景
*   **核心问题**：现有的胸部 X 射线（CXR）视觉-语言预训练（VLP）模型主要存在三个局限：
    1.  **多视图利用不足**：忽视了胸片通常包含正位（Frontal）和侧位（Lateral）的特性，或仅进行简单的统一特征融合，忽略了不同视图对病理诊断的差异化贡献。
    2.  **掩码机制盲目性**：传统的随机掩码语言建模（MLM）在医疗场景下效率低下，未能聚焦于报告中关键的病理描述。
    3.  **全局对齐的假阴性问题**：对比学习通常将不同患者的影像视为负样本，但在医学上，患有相同疾病的不同患者应具有语义相似性，强行推开会导致表征学习受阻。

### 2. 论文提出的方法论
该框架通过三个核心模块实现高效的表征学习：
*   **关键语义增强的多视图 MLM (Key Semantics-enhanced Multi-view MLM)**：
    *   该模块不再随机掩码，而是识别并聚合跨视图的病理相关关键补丁（Patches）。
    *   利用图像中的显著性区域来指导文本掩码的预测，确保模型能够学习到影像特征与核心医学术语之间的深层映射。
*   **正侧位对齐模块 (Frontal-Lateral Alignment, FLA)**：
    *   独立提取正位和侧位视图的病理特征。
    *   在聚合过程中既保留视图特有的信息，又通过一致性约束确保两者在语义空间的一致性，减少关键信息的丢失。
*   **高阶语义对齐 (High-order Semantic Alignment, HSA)**：
    *   引入**原型对齐（Prototype-level alignment）**。通过将图像和文本特征与一组可学习的语义聚类中心（原型）进行对齐，缓解“假阴性”问题。
    *   即使是不同患者的样本，只要其病理特征相似，就会被映射到相近的原型空间。
