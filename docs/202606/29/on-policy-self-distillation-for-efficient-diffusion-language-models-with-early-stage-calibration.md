### 论文总结：面向高效扩散语言模型的带早期校准的同策略自蒸馏 (COPSD)

#### 1. 论文的核心问题与整体含义
扩散语言模型（DLLMs）通过并行生成 token 展现出超越自回归模型（ARMs）的潜力。然而，DLLMs 在生成初期面临严重的**早期预测不准**问题：由于此时大部分 token 仍被掩码，上下文信息极度匮乏，导致初始预测偏差较大。这种早期错误会随采样过程传播，不仅降低了生成质量，还限制了推理效率（即无法在保持准确性的前提下增加每步并行生成的 token 数量）。本研究旨在通过校准早期 token 的预测分布，提升 DLLMs 的生成质量与并行推理效率。

#### 2. 论文提出的方法论：COPSD
论文提出了**校准同策略自蒸馏（Calibrated On-Policy Self-Distillation, COPSD）**，其核心思想是利用模型在后期解码步骤（上下文更丰富、预测更准）的信息来指导早期步骤的训练。

*   **核心思想**：将后期步骤生成的更准确的 token 分布作为“伪标签”，通过自蒸馏的方式将其知识传递给早期步骤，使模型在上下文较少时也能做出更具前瞻性的预测。
*   **关键技术细节**：
    *   **校准分布（Calibrated Distribution）**：定义了一个未归一化的目标分布，它结合了当前步 $t$ 的预测和未来步 $t-s$ 的预测。通过衡量两步之间分布的距离（$L_2$ 范数），对早期预测进行修正。
    *   **同策略自蒸馏**：使用逆 KL 散度作为损失函数，让模型学习上述校准后的分布。为了稳定训练，使用模型之前的参数版本（$\theta'$）来提供目标分布。
    *   **组相对归一化（Group Relative Normalization）**：借鉴了强化学习中的 GRPO 算法，对校准项进行组内归一化（减均值除以标准差）。这一步至关重要，因为它解决了由于采样样本有限导致的训练高方差和模型崩溃问题。
*   **算法流程**：从当前模型采样响应 -> 对响应进行不同比例的掩码（模拟早期 $t$ 和后期 $t-s$ 状态） -> 计算校准后的目标分布 -> 应用组归一化损失函数更新模型参数。
