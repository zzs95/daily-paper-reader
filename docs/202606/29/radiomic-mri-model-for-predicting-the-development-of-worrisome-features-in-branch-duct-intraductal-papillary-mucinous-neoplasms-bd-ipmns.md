*   **研究背景**：分支胰管型导管内乳头状黏液性肿瘤（BD-IPMNs）具有从良性向恶性演变的潜力。虽然最新的“2024京都指南”定义了“令人担忧的特征”（WF）和“高风险征象”（HRS）来指导临床，但在患者初次检查时预测其未来是否会产生这些危险特征依然极具挑战。

*   **核心思想**：通过提取 MRI 图像中高维度的定量特征（肉眼难以识别的纹理和几何信息），利用机器学习算法构建预测模型。

    *   **验证方式**：采用**五折交叉验证（5-fold cross-validation）**来评估模型的稳健性和泛化能力。
