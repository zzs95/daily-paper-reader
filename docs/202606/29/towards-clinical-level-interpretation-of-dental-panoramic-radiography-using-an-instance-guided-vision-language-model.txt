## Data availability

The Dual-labeled Panoramic Radiographs Dataset and the Teeth Segmentation on Dental X-ray Images dataset are available at [https://www.kaggle.com/datasets/zwbzwb12341234/a-dual-labeled-dataset](https://www.kaggle.com/datasets/zwbzwb12341234/a-dual-labeled-dataset) and [https://www.kaggle.com/datasets/humansintheloop/teeth-segmentation-on-dental-x-ray-images](https://www.kaggle.com/datasets/humansintheloop/teeth-segmentation-on-dental-x-ray-images). The multi-hospital clinical datasets used in this study are not publicly available because they contain protected patient information and are subject to institutional data-sharing agreements and ethical restrictions. De-identified data may be made available for non-commercial research purposes from the corresponding author upon request, subject to institutional approval and data-use agreements. Requests for commercial use will not be considered. [Source data](https://www.nature.com/articles/s41551-026-01713-8#Sec37) are provided with this paper.

## Code availability

To ensure reproducibility, we provide detailed implementation notes while preserving readability for both clinical and broader scientific audiences. The source code for DentFound is openly available via GitHub at [https://github.com/ahukui/DentFound](https://github.com/ahukui/DentFound).
