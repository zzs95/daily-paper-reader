*   **研究背景**：牙科全景放射摄影（DPR）是口腔临床诊断的基石，但面临巨大的解读压力。由于放射科医生短缺，许多临床报告往往只关注患者的“主诉”区域，而忽略了全口范围内的其他潜在病变，导致漏诊率高。
*   **核心问题**：现有的通用视觉语言模型（VLM）在处理牙科影像时，难以实现对32颗甚至更多牙齿（含乳牙）的细粒度定位与精准病理描述。
