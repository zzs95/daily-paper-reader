以下是对该论文的深度总结：

### 1. 核心问题与整体含义
*   **研究背景**：胸部 X 射线（CXR）检查通常包含正位（Frontal）和侧位（Lateral）两个视图，并配有医生的文字报告。
*   **核心问题**：
    1.  **多视图利用不足**：现有方法要么只看单视图，要么对多视图进行简单的统一特征融合，忽略了不同视图对特定病理诊断的贡献差异。
    2.  **跨模态交互效率低**：传统的掩码语言建模（MLM）采用随机掩码，缺乏与图像病理区域的深度关联。
    3.  **全局对齐的局限性**：传统的对比学习（Global Alignment）存在“假阴性”问题，即语义相似的不同样本被错误地推开，影响表征质量。

### 2. 方法论
论文提出了一个包含三个核心组件的预训练框架：
*   **语义增强的多视图 MLM (Key Semantics-enhanced Multi-view MLM)**：
    *   该模块不再随机掩码文本，而是识别并聚合跨视图中与病理相关的关键图像补丁（Patches）。
    *   利用这些富含语义的视觉特征作为监督信号来指导文本掩码的恢复，从而强化视觉与文本之间的细粒度交互。
*   **正侧位对齐模块 (Frontal-Lateral Alignment, FLA)**：
    *   独立提取正位和侧位视图的病理特征，以保留视图特有的关键信息。
    *   通过一致性约束确保在特征聚合过程中，关键的病理语义不被稀释或丢失。
*   **高阶语义对齐 (High-order Semantic Alignment, HSA)**：
    *   引入“原型（Prototypes）”概念，将图像和文本特征与一组学习到的语义聚类中心进行对齐。
    *   这种原型级别的对齐缓解了样本间的假阴性问题，从更高阶的语义维度增强了全局对齐的效果。
