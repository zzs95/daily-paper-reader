*   **核心问题**：低剂量CT（LDCT）是肺癌筛查的标准手段，但其高噪声和伪影会降低诊断准确性。目前学术界大多关注如何“去噪”（LDCT $\rightarrow$ SDCT），但高质量标注的真实LDCT数据集非常稀缺。

### 2. 论文提出的方法论
论文对比了三种不同维度的图像退化技术，旨在将 SDCT 转换为具有 LDCT 特性的合成图像：
*   **方法1：简单正弦图域模拟（Sinogram-domain Simulation）**
    *   核心思想：利用 Radon 变换将图像转回正弦图空间，注入泊松噪声（模拟量子噪声）和高斯噪声（模拟电子噪声），再通过反 Radon 变换重建。
*   **方法2：基于物理模型的 Pix2Pix 复制（Physics-based via Pix2Pix）**
    *   核心思想：利用条件生成对抗网络（Pix2Pix）学习一种已验证的物理噪声模型（考虑了领结滤波器、管电流调制等复杂因素）。由于该物理模型需要不可获取的原始投影数据，作者通过配对的图像数据训练 Pix2Pix 来“模拟”这一物理退化过程。
*   **方法3：非配对循环生成对抗网络（CycleGAN）**
    *   核心思想：利用非配对的 SDCT 和 LDCT 图像，通过循环一致性损失学习两个领域之间的分布映射，直接在图像域生成具有 LDCT 统计特性的伪图像。

**后续流程**：生成合成数据后，提取 851 个影像组学特征（Radiomics），经过稳定性分析、冗余去除和特征选择（LASSO, RFE等），最后输入机器学习分类器。

### 6. 论文的主要结论与发现
