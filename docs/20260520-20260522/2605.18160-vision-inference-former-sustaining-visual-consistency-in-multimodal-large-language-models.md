### 1. 核心问题与研究动机
论文指出了当前多模态大语言模型（MLLM）主流“连接器（Connector）”范式的两大局限性：
*   **视觉地位被稀释**：视觉信息虽是核心证据，但在模型中被视为与文本 Token 等同的输入，削弱了视觉模态的独特贡献。
*   **视觉一致性衰减（Visual Consistency Decay）**：随着生成长度的增加（尤其是在有限的上下文窗口内），模型对初始视觉 Token 的依赖逐渐减弱，转而过度依赖文本先验（Linguistic Priors），导致生成内容偏离图像事实，产生幻觉。
**核心目标**：建立一种在推理过程中能持续注入视觉语义的机制，确保模型在整个生成周期内始终锚定在视觉内容上。

### 2. 方法论：Vision Inference Former (VIF)
VIF 是一个轻量级的辅助模块，旨在解码阶段动态地将视觉特征注入 LLM 的输出空间。
*   **核心思想**：不只是在输入端一次性喂入图像，而是在每一个解码步（Decoding Step）都根据当前的隐藏状态动态检索相关的视觉信息。
*   **关键架构**：VIF 由一个两层 Transformer 组成：
    1.  **内视觉自注意力层（Intra-visual Self-attention）**：对原始视觉特征进行精炼，捕捉图像内部的语义依赖。
    2.  **跨模态交叉注意力层（Cross-attention）**：以 LLM 当前步的隐藏状态 $h_l$ 作为查询（Query），以精炼后的视觉特征作为键（Key）和值（Value），提取当前上下文最相关的视觉证据。
*   **融合策略**：采用简单的加法融合 $h'_l = \text{Norm}(Z_h + h_l)$，将 VIF 输出的视觉增强表示注入到 LLM 的预测头之前。
*   **信息论解释**：通过引入动态变量 $A_l$，增加了输出 Token 与视觉模态之间的互信息，从理论上解释了为何能缓解一致性衰减。
