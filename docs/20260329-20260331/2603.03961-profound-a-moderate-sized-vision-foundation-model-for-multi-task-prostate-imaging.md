*   **临床背景**：前列腺癌的诊断和治疗高度依赖多参数磁共振成像（mpMRI），但其解读极度依赖放射科专家的经验，难以大规模推广。
*   **技术瓶颈**：现有的深度学习模型大多是“任务特定型”，需要大量标注数据，且泛化能力差。虽然通用医疗基础模型（如 RadFM）已出现，但由于前列腺 mpMRI 协议高度专业化且标准化时间较短，通用模型在这一特定领域的表现往往不足。
*   **研究目标**：开发一个专门针对前列腺 mpMRI 的中等规模视觉基础模型（ProFound），旨在通过自监督预训练学习通用的前列腺体积表示，从而在数据稀缺的情况下高效适配多种临床下游任务。

### 2. 方法论：核心思想与技术细节
*   **核心思想**：采用**掩码自编码器（MAE）**框架，在无标注的大规模前列腺 mpMRI 数据上进行自监督预训练，使模型能够从局部可见区域重建缺失的解剖结构和病变特征。
*   **关键技术细节**：
    *   **多模态输入**：将 T2w、高 b 值 DWI 和 ADC 三种序列堆叠为通道，形成 3D mpMRI 输入。
    *   **两种架构变体**：
        1.  **ProFound-ViT**：基于 ViT-B 的 3D MAE，将体积划分为非重叠立方块（patches），使用 Transformer 块处理可见块。
        2.  **ProFound-Conv**：基于 ConvNeXtV2 的全卷积 3D MAE，使用稀疏卷积直接在空间域操作，对掩码区域进行重建。
    *   **预训练目标**：通过高比例掩码（ViT 为 0.75，Conv 为 0.6）强制模型学习深层的空间和模态间关联。
    *   **下游适配**：支持线性探测（冻结编码器）和全参数微调（使用层级学习率衰减 LLRD）。
