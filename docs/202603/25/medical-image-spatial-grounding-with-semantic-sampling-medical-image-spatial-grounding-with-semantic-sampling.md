### 1. 核心问题与研究背景
*   **核心问题**：现有的视觉语言模型（VLM）在处理三维医学影像（如 CT、MRI）时，虽然能生成包含医学术语的报告，但在**空间定位（Spatial Grounding）**方面表现极差，经常出现“一本正经地胡说八道”（即无法将术语准确对应到图像中的空间坐标）。
*   **研究背景**：医学影像具有独特的三维属性、复杂的解剖学坐标系（如 RAS 坐标）以及专业的方位术语（如“Superior/Inferior”与“Above/Below”的区别）。现有的通用 VLM 往往依赖语言先验而非视觉事实，导致在临床高风险场景下缺乏实用性。

### 2. 方法论：核心思想与关键技术
论文提出了两个核心贡献：
*   **MIS-Ground 评测框架**：一个专门用于测试 VLM 对 3D 医学影像空间理解能力的基准。它涵盖了图像模态、切片方向、坐标系统、解剖学术语与口语化术语的对比，以及视觉提示（点、框、掩码）的影响。
*   **MIS-SemSam（语义采样）**：
    *   **核心思想**：针对医学术语在解码时的“脆弱性”（微小扰动可能导致方位词反转），在推理阶段引入一种**模型无关（Model-agnostic）**的优化策略。
    *   **技术细节**：
        1.  **离线构建语义邻域**：在 token 嵌入空间中，为所有内容 token 预计算 K 个最近邻。
        2.  **推理时语义重评分**：在生成下一个 token 时，不只看单个 token 的概率，而是聚合其语义邻域内的概率质量。如果一个概念（如“Superior”）在语义邻域内有广泛的概率支持，则该 token 会被优先选择。
        3.  **稳定性**：这种方法能有效防止模型在长文本推理（CoT）中因一个局部错误的 token 而导致后续逻辑崩塌。

*   **术语偏好**：模型在标准视角下偏好解剖术语，但在非标准存储模式下会产生混乱。

*   **低成本优化**：MIS-SemSam 不需要重新训练模型，仅靠推理时的语义聚合就能显著提升性能，具有极高的实用价值。
*   **基准测试的严谨性**：MIS-Ground 填补了医学 VLM 缺乏空间定位评测的空白，考虑到了 3D 影像特有的切片方向和坐标系问题。
*   **深入的消融分析**：成功揭示了 VLM 在医学影像中过度依赖“解剖学常识”而非“实际视觉内容”的弊端。
