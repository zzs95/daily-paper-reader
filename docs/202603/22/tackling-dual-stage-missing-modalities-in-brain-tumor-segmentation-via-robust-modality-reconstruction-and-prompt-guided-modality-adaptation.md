*   **研究背景**：现有方法大多假设“训练阶段模态完整，仅推理阶段模态缺失”。但在实际临床场景中，训练集本身往往也存在大量缺失模态的数据（即**双阶段缺失**）。

### 2. 方法论：核心思想与技术细节
论文提出了一个名为 **RP-Net** 的框架，包含两个核心模块：

*   **鲁棒模态重建 (Robust Modality Reconstruction, RMR)**：
    *   **目的**：在训练和推理阶段，利用现有的模态去预测/重建缺失的模态。
    *   **技术细节**：采用基于生成对抗网络（GAN）的思想，通过一个共享编码器提取公共特征，再通过模态特定的解码器重建缺失图像。为了增强鲁棒性，引入了特征解耦机制，确保重建过程能够捕捉到模态间的共性与特性。
*   **提示引导的模态自适应 (Prompt-guided Modality Adaptation, PMA)**：
    *   **目的**：让分割网络感知当前的模态组合状态，动态调整特征提取策略。
    *   **技术细节**：引入了“模态提示词（Modality Prompts）”。这些提示词作为可学习的参数，编码了不同模态的存在状态信息。通过将提示信息注入到分割网络的编码器中，引导模型在不同模态缺失组合下实现自适应的特征融合。
*   **算法流程**：
    1.  输入任意组合的模态。
    2.  RMR 模块补全缺失模态，生成伪完整模态组。
    3.  PMA 模块根据原始模态的缺失掩码生成提示向量。
    4.  分割网络结合补全后的图像与提示向量，输出最终的肿瘤分割图。
