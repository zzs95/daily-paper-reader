自动医学报告生成（MRG）旨在根据医学图像（如胸部X光片）自动撰写诊断描述。
*   **研究动机**：尽管检索增强（Retrieval-Augmented）策略在MRG中表现出色，但现有方法存在两个主要缺陷：
    1.  **单模态检索局限性**：大多依赖“图像找图像”或“文本找文本”，难以充分利用图像与文本之间的跨模态语义关联。
    2.  **忽略不确定性**：检索到的参考内容可能包含噪声或与当前病例不完全匹配，现有模型往往盲目信任检索结果，导致生成的报告出现误导性信息。
*   **核心目标**：通过引入跨模态检索和不确定性建模，提升报告生成的准确性、鲁棒性和临床可靠性。

### 2. 论文提出的方法论
论文提出了一种**不确定性感知跨模态检索（UACMR）**框架，其核心流程如下：
*   **跨模态检索机制**：不同于传统的图像检索，该方法直接学习图像特征与报告文本特征的共同嵌入空间。在生成阶段，利用当前图像作为查询（Query），从历史数据库中检索语义最相关的报告文本作为参考。
*   **不确定性建模（Uncertainty-Awareness）**：
    *   引入概率建模来评估检索到的参考信息与当前图像之间的匹配置信度。
    *   通过量化检索过程中的“不确定性”，模型可以动态调整对检索信息的依赖程度：当检索结果高度确定时，加强参考信息的引导；当不确定性较高时，则更多依赖图像本身的视觉特征。
*   **增强生成器**：将视觉特征、检索到的文本特征以及不确定性权重共同输入到基于 Transformer 的解码器中，生成最终的医学报告。
