*   **核心问题**：由于单目视觉缺乏显式的深度信息，深度估计存在固有的“病态”歧义性（即一个 2D 像素点可能对应 3D 空间中无数个点），这导致物体在 3D 空间中的定位精度远低于双目或激光雷达（LiDAR）方案。
*   **研究动机**：探索如何更有效地利用深度预测信息来增强图像特征，通过多尺度融合机制缓解深度不确定性，从而提升检测性能。

### 2. 论文提出的方法论
*   **核心思想**：提出 **MonoDGD** 框架，将深度图作为一种“引导信号”而非简单的输入，通过多尺度融合机制将深度特征与语义特征深度集成。
*   **关键技术细节**：
    *   **端到端架构**：采用统一的网络同时进行深度预测和目标检测，实现任务间的特征共享与协同优化。
    *   **深度预测分支**：从骨干网络提取特征，生成密集的深度图。
    *   **多尺度深度引导融合（Multi-scale Depth-Guided Fusion）**：在特征提取的不同阶段，利用预测的深度信息对图像特征进行加权或重构。这种机制能让网络在局部细节（近处物体）和全局上下文（远处物体）之间取得平衡。
    *   **3D 检测头**：基于融合后的增强特征，回归物体的中心点、深度偏移、尺寸和旋转角。
