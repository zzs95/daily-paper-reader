### 2. 论文提出的方法论
该论文提出了一种**物体-场景-相机（Object-Scene-Camera）分解与重组框架**，核心思想如下：
*   **分解（Decomposition）**：将原始图像及其标注信息拆解为三个独立维度：
    1.  **物体（Object）**：提取前景物体的几何属性（尺寸、类别）和纹理特征。
    2.  **场景（Scene）**：提取背景环境的视觉上下文。
    3.  **相机（Camera）**：提取相机的内参（焦距、主点）和外参（安装高度、角度）。
*   **重组（Recomposition）**：通过数学变换和图像合成技术，将不同来源的物体、场景和相机参数进行交叉组合。
    *   例如：将 A 场景中的车辆“放置”到 B 场景中，并模拟 C 相机的成像参数。
*   **关键技术细节**：这种重组机制不仅是简单的图像剪贴，还涉及几何一致性的对齐，确保生成的合成样本在物理逻辑上（如地平面接触、透视关系）是合理的，从而为模型提供多样化且符合物理规律的训练样本。
