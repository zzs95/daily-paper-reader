### 1. 核心问题与整体含义（研究动机和背景）
*   **核心问题**：现有的放射学报告自动评估指标（如 BLEU、ROUGE 等词汇重叠指标）与临床真实价值脱节，无法有效处理临床语言中的否定、歧义以及识别低流行率（罕见）的异常表现。
*   **研究背景**：虽然 CheXbert 和 RadGraph 等领域特定指标有所进步，但它们的标签空间有限（如 CheXbert 仅涵盖 14 类），且在处理复杂临床逻辑和罕见病变时表现不佳。这导致 AI 生成的报告在技术指标上得分高，但在临床实践中缺乏可信度。

### 2. 论文提出的方法论
*   **核心思想**：开发一个“临床医生引导的人机协作框架”，利用大语言模型（LLM）的推理能力，结合放射科专家的知识，对报告进行多标签提取，并以此定义 **Ran Score**（基于宏平均 F1 分数的评估指标）。
*   **关键技术细节**：
    *   **标准化分类体系**：构建了包含 21 类异常的标准化标签空间（比 CheXbert 更广，新增了气管支气管异常、空洞、纵隔异常等）。
    *   **迭代提示词优化（Error-driven Prompt Refinement）**：不通过微调模型，而是由放射科医生对 LLM 的错误进行分析，针对性地在 Prompt 中加入核心术语、消除歧义的说明以及正负示例（Few-shot）。
    *   **Ran Score 定义**：使用优化后的 LLM 提取器，分别从“生成报告”和“参考报告”中提取 21 类标签，计算两者之间的**宏平均 F1 分数（Macro-averaged F1）**。该指标对低流行率病变更敏感。
