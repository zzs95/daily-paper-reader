## 摘要
尽管大语言模型（LLMs）在 CT 报告生成方面取得了进展，但现有方法通常对 3D 体积进行整体编码，无法区分关键信息线索与冗余的解剖背景。受放射学认知减影（radiological cognitive subtraction）的启发，我们提出了差异化视觉提示（DiffVP），它将报告生成建立在显式的、高层级的扫描与参考图像之间的语义差异上，而非仅仅依赖于绝对视觉特征。DiffVP 采用分层差异提取器，将互补的全局和局部语义差异捕捉到共享潜空间中，并结合差异到提示生成器，将这些信号转换为可学习的视觉前缀标记，用于 LLM 的条件引导。这些差异提示作为结构化的条件信号，隐式地抑制了不变的解剖结构，同时放大具有诊断相关性的视觉证据，从而在无需显式病灶定位的情况下促进准确的报告生成。在两个大规模基准测试中，DiffVP 一致优于先前的方法，平均 BLEU-1-4 分别提高了 +10.98 和 +4.36，并进一步提升了在 RadGenome-ChestCT 上的临床效能（F1 分数为 0.421）。所有代码将在 https://github.com/ArielTYH/DiffVP/ 发布。

## Abstract
While large language models (LLMs) have advanced CT report generation, existing methods typically encode 3D volumes holistically, failing to distinguish informative cues from redundant anatomical background. Inspired by radiological cognitive subtraction, we propose Differential Visual Prompting (DiffVP), which conditions report generation on explicit, high-level semantic scan-to-reference differences rather than solely on absolute visual features. DiffVP employs a hierarchical difference extractor to capture complementary global and local semantic discrepancies into a shared latent space, along with a difference-to-prompt generator that transforms these signals into learnable visual prefix tokens for LLM conditioning. These difference prompts serve as structured conditioning signals that implicitly suppress invariant anatomy while amplifying diagnostically relevant visual evidence, thereby facilitating accurate report generation without explicit lesion localization. On two large-scale benchmarks, DiffVP consistently outperforms prior methods, improving the average BLEU-1-4 by +10.98 and +4.36, respectively, and further boosts clinical efficacy on RadGenome-ChestCT (F1 score 0.421). All codes will be released at https://github.com/ArielTYH/DiffVP/.

*   **核心问题**：现有的基于大语言模型（LLM）的 CT 报告生成方法通常将 3D 图像体积作为一个整体进行编码。然而，3D CT 图像中异常区域极其稀疏，大部分是冗余的解剖背景（如骨骼、肌肉）。LLM 难以从海量背景噪声中自主识别出关键的病理线索。
*   **研究动机**：受放射科医生临床诊断逻辑的启发——医生通常会将当前扫描图像与“正常解剖结构”进行对比（认知减法），从而识别出细微的偏差。论文旨在通过引入“正常参考图像”作为先验，显式建模语义差异，引导 LLM 关注临床相关的病变区域。

### 2. 方法论
*   **核心思想**：提出 **DiffVP（差异化视觉提示）** 框架，不直接进行像素级减法，而是在高层语义空间提取扫描图像与参考图像的差异，并将其转化为 LLM 的可学习前缀提示（Prefix Tokens）。
*   **关键技术细节**：
    *   **分层差异提取器 (HDE)**：
        *   **全局增量算子 (Global Delta)**：利用共享的视觉重采样器（Visual Resampler）将图像投影到统一潜空间，通过 Transformer 层捕捉整体结构和语义的偏移。
        *   **局部增量算子 (Local Delta)**：基于欧几里得距离计算标记（Token）间的权重，聚合残差向量，捕捉细微的局部病变。
    *   **差异到提示生成器 (DPG)**：将提取的全局和局部差异融合，通过投影层转换为 $p$ 个视觉前缀标记（$\Delta_{prompt}$），直接插入 LLM 的输入序列。
    *   **诊断语义引导 (E)**：利用预训练的分类器预测 18 类疾病标签，作为辅助语义锚点。
*   **算法流程**：输入目标 CT 和参考 CT $\rightarrow$ 视觉编码与重采样 $\rightarrow$ 提取分层语义差异 $\rightarrow$ 生成视觉差异提示 $\rightarrow$ 结合诊断标签输入 LoRA 微调后的 LLM $\rightarrow$ 生成报告。

### 6. 主要结论与发现
*   **性能领先**：DiffVP 在两个数据集上均显著优于现有方法，BLEU-4 指标在 RadGenome 上提升了约 37.8%。
*   **临床准确性**：在临床有效性（CE）测试中，F1 分数达到 0.421，远超之前的模型，证明其对病灶的捕捉更敏锐。
*   **语义差异优于像素差异**：实验证明在潜空间建模语义差异比直接进行图像像素减法更具鲁棒性，能有效抵御配准误差。

### 7. 优点
*   **临床一致性**：方法论紧贴放射科医生的实际工作流，具有很强的可解释性。
*   **鲁棒性强**：通过分层语义建模，解决了 3D 图像配准难、解剖结构个体差异大的问题。
*   **即插即用**：作为一种提示工程（Prompting）方法，可以较容易地迁移到不同的 LLM 主干网络上。

*   **计算开销**：虽然使用了轻量化适配，但推理时需要同时处理目标和参考两组图像特征，增加了显存占用。
