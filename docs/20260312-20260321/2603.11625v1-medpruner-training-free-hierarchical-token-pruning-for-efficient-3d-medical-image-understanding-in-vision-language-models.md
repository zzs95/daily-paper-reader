## 摘要
虽然专门的医学视觉语言模型（VLMs）在解释 2D 和 3D 医学模态方面取得了显著成功，但它们在 3D 体数据上的部署仍受限于严重的计算低效。当前的架构通常由于直接拼接连续的 2D 切片而存在大量的解剖冗余，并且缺乏使用固定剪枝率处理不同切片间异构信息密度的灵活性。为了应对这些挑战，我们提出了 MedPruner，这是一个无需训练且与模型无关的分层 Token 剪枝框架，专门为高效的 3D 医学图像理解而设计。MedPruner 引入了一个两阶段机制：一个基于锚点的切片间过滤模块，用于消除切片级的时序冗余；随后是一个动态信息核选择策略，通过量化累积注意力权重来实现自适应的 Token 级压缩。在三个 3D 医学基准和三种不同的医学 VLM 上进行的广泛实验揭示了现有架构中存在巨大的 Token 冗余。值得注意的是，MedPruner 使 MedGemma 等模型在仅保留不到 5% 的视觉 Token 的情况下，仍能保持甚至超过其原始性能，从而大幅降低了计算开销，并验证了动态 Token 选择对于实际临床部署的必要性。我们的代码将会发布。

## Abstract
While specialized Medical Vision-Language Models (VLMs) have achieved remarkable success in interpreting 2D and 3D medical modalities, their deployment for 3D volumetric data remains constrained by significant computational inefficiencies. Current architectures typically suffer from massive anatomical redundancy due to the direct concatenation of consecutive 2D slices and lack the flexibility to handle heterogeneous information densities across different slices using fixed pruning ratios. To address these challenges, we propose MedPruner, a training-free and model-agnostic hierarchical token pruning framework specifically designed for efficient 3D medical image understanding. MedPruner introduces a two-stage mechanism: an Inter-slice Anchor-based Filtering module to eliminate slice-level temporal redundancy, followed by a Dynamic Information Nucleus Selection strategy that achieves adaptive token-level compression by quantifying cumulative attention weights. Extensive experiments on three 3D medical benchmarks and across three diverse medical VLMs reveal massive token redundancy in existing architectures. Notably, MedPruner enables models such as MedGemma to maintain or even exceed their original performance while retaining fewer than 5% of visual tokens, thereby drastically reducing computational overhead and validating the necessity of dynamic token selection for practical clinical deployment. Our code will be released.

*   **研究背景**：随着医学视觉语言模型（VLM）从 2D 转向 3D（如 CT、MRI），由于 3D 影像由大量连续切片组成，直接拼接这些切片的视觉 Token 会导致“Token 爆炸”，超出大语言模型（LLM）的上下文窗口并大幅增加计算开销。
*   **核心问题**：
    1.  **解剖冗余**：相邻切片之间存在极高的空间相似性。
    2.  **信息密度不均**：不同切片的重要性差异巨大（如病灶切片 vs. 背景切片），现有的固定比例剪枝方法无法灵活处理这种异构性。

### 2. 论文提出的方法论
MedPruner 是一个**无需训练（Training-free）**且**模型无关**的分层 Token 剪枝框架，包含两个核心阶段：

*   **阶段一：基于锚点的切片间过滤（Inter-slice Anchor-based Filtering, IAF）**
    *   **核心思想**：通过动态检测切片间的形态变化来消除时序/空间冗余。
    *   **技术细节**：维护一个动态“锚点切片”，计算当前切片与锚点切片的像素级平均 L1 距离。若距离超过预设阈值 $\gamma$，则认为该切片包含显著的新解剖特征，将其保留并更新为新锚点；否则将其剔除。
*   **阶段二：动态信息核选择（Dynamic Information Nucleus Selection, DINS）**
    *   **核心思想**：在保留的切片内部，根据注意力分布动态选择最重要的 Token。
    *   **技术细节**：
        1.  从视觉编码器的自注意力层提取注意力分数，进行多头平均和温度缩放归一化。
        2.  借鉴“核采样（Nucleus Sampling）”思路，将 Token 按重要性降序排列，动态选择累积注意力权重达到阈值 $\tau$ 的最小 Token 集合（称为 Primary Tokens）。
        3.  对未选中的冗余 Token 进行二分匹配聚类，以保留全局上下文而不增加序列长度。

### 6. 主要结论与发现
*   **极高的冗余度**：3D 医学影像中存在巨大的 Token 冗余。MedPruner 在 **MedGemma 模型上仅保留不到 5%（甚至低至 2.46%）** 的视觉 Token，性能依然能持平甚至超过原始模型。

*   **动态选择的必要性**：实验证明固定比例剪枝（如 HiPrune）在医学场景下效果不如基于累积权重的动态选择，因为医学影像的信息分布极度不均。

### 7. 优点
*   **高效且通用**：无需重新训练模型，可直接插入现有的各种 VLM 架构中，极大地降低了部署门槛。
*   **动态自适应**：通过“核选择”策略，能够根据切片内容的复杂程度自动调整保留的 Token 数量，比固定比例剪枝更科学。
*   **去噪能力**：不仅减少了计算量，还通过剔除无关背景提升了模型对病灶等关键信息的理解能力。

### 8. 不足与局限
*   **依赖视觉编码器质量**：DINS 策略高度依赖于视觉编码器产生的注意力图，如果编码器本身未经过良好的医学预训练，其注意力分布可能无法准确反映临床重要性。
*   **切片过滤算法较简单**：IAF 模块使用的像素级 L1 距离虽然高效，但可能无法捕捉到细微的语义变化（例如微小病灶的出现可能在像素差异上并不显著）。
*   **应用范围限制**：目前主要在 CT 和 MRI 等体积数据上验证，对于具有更高时序动态性的医学视频（如超声、内窥镜）的适用性尚待进一步研究。
