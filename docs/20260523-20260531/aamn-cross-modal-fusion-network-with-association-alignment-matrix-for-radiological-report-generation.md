### 2. 论文提出的方法论
论文提出了 **AAMN（Association Alignment Matrix Network）**，其核心架构包含两个关键模块：
*   **视觉特征对齐模块 (VFAM)**：
    *   **核心思想**：针对同一患者的正面（Front）和侧面（Side）图像，提取各自的视觉特征。

*   **跨模态特征融合模块 (CFFM)**：
    *   **核心思想**：优化视觉表征与文本生成器之间的信息传递。
    *   **技术细节**：利用注意力机制，将对齐后的视觉特征与文本序列进行交互，增强模型在生成特定医学术语时对图像关键区域的关注度，确保“所见即所言”。
