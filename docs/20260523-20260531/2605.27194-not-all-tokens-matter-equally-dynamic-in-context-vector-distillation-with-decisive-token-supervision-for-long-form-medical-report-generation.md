### 1. 核心问题与整体含义（研究动机和背景）
论文关注的是**多模态长文本生成**（特别是医学影像报告生成，MRG）中的上下文学习（ICL）蒸馏问题。
*   **研究动机**：虽然将示例效应蒸馏为隐空间干预（如上下文向量 ICV）在短文本任务（如VQA）中很有效，但在长文本生成中面临两大挑战：
    1.  **Token重要性不均（决定性Token受监督不足）**：医学报告中充斥着大量高频模板词（如“the lungs are clear”），而真正决定质量的病理词（如“pneumothorax”）和结束符（EOS）分布稀疏。标准交叉熵损失平均对待所有Token，导致关键信息被淹没。
    2.  **自回归轨迹漂移**：静态的干预向量在长程生成中，会随着模型自回归解码偏离教师强制（Teacher-forced）轨迹而失效。
*   **核心问题**：如何使蒸馏后的轻量级干预在长文本生成中保持临床忠实度并实现准确的终止。

### 2. 方法论：DIVE 框架
论文提出了 **DIVE**（Dynamic In-context VEctor Distillation），其核心思想是通过动态适配和非均匀监督来增强模型。
*   **状态条件动态引导（State-conditioned Dynamic Steering）**：
    *   放弃了传统的固定残差向量，改用轻量级的瓶颈 MLP 适配器（Bottleneck MLP Adapters）。
    *   该适配器根据当前解码步骤的隐藏状态 $h_t$ 动态计算注入的残差信号 $\delta_t$，从而应对自回归过程中的轨迹漂移。
*   **决定性标记监督（Decisive-token Supervision）**：
    *   **病理词加权**：利用领域先验词典（基于 CheXpert 的 14 类疾病及其临床表达）识别目标文本中的病理相关 Token，并在计算交叉熵（CE）损失时赋予更高权重。
    *   **EOS 事件加权**：显著提升 EOS 标记的损失权重，以解决模型在长文本生成中常见的“停不下来”或过早终止的校准问题。
*   **蒸馏目标函数**：结合了 Top-K KL 散度（模仿教师模型的概率分布）和加权交叉熵损失（$L_{CE}$），在冻结底座模型参数的前提下，仅优化适配器参数。
